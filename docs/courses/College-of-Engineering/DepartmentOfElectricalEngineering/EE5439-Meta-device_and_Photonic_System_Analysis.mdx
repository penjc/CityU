# EE5439
## 元器件与光子系统分析

## **📘 简介**
本课程为学生提供光电子与元器件系统的理论知识、仿真建模和分析技能。学生将学习天线设计、电磁波与物质的相互作用、纳米结构元器件与光子系统的特性分析。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5439.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5439.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 解释电磁波和元器件与光子系统的基本原理；

✔️ 应用光子系统分析技术解决实际问题；

✔️ 设计并分析元器件与光子系统，验证仿真结果；

✔️ 通过独立研究发现元器件与光子系统的潜在应用与创新。

# EE5410
## 信号处理

## **📘 简介**
本课程旨在为学生提供信号处理的坚实基础，并帮助学生利用信号处理技术解决实际问题。课程涵盖连续时间和离散时间信号、系统分析及数字滤波器的设计与实现。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5410.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5410.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别连续时间和离散时间信号与系统的特性；

✔️ 解释信号处理中不同变换之间的关系；

✔️ 分析离散时间系统并利用适当的变换计算系统参数；

✔️ 设计和实现数字滤波器；

✔️ 应用信号处理技术解决科学与工程问题。

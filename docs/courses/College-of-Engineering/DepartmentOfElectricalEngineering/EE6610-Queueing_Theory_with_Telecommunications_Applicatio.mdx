# EE6610
## 排队论与电信应用

## **📘 简介**
本课程旨在教授学生概率与排队模型的理论、仿真及数值算法，并将其应用于现代和未来电信流量与网络建模。学生将学习如何利用这些模型提供具成本效益的电信服务。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 将概率与排队论应用于实际问题；

✔️ 从基本原理开发关键性能指标的分析结果；

✔️ 编程模拟排队系统并评估置信区间；

✔️ 计算通信网络的链路容量，并分析网络性能。

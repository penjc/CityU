# EE5805
## Java 网络编程

## **📘 简介**
本课程旨在让学生理解面向对象设计和互联网应用开发的概念与技术。Java 作为主要的面向对象编程语言被用作教学和实现工具。

🔗 **相关链接**

- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5805.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 使用 Java 语言解决通用计算问题；

✔️ 识别 Java 网络应用的通信模型和架构；

✔️ 实现可在内网和互联网运行的 Java 网络应用。

# EE5809
## 数字音频处理及其应用

## **📘 简介**
本课程旨在为学生提供数字音频处理及其应用的坚实基础，培养学生在音频工程职业所需的技能。课程涵盖音频信号的特性、现代音频编码标准及多通道音频系统设计。

- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5809.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述音频信号特性及过采样模数转换的原理；

✔️ 展示时域与频域音频信号的数字编码技能；

✔️ 解释现代音频编码标准的概念及其内部运作；

✔️ 处理和合成音乐信号；

✔️ 设计多通道家庭音频系统并评估其性能。

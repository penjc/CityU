# EE6802
## 数字信号处理进阶

## **📘 简介**
本课程提供高级数字信号处理的理论与实践知识，特别关注多速率信号处理技术及其系统设计。学生将学习离散傅里叶变换（DFT）、快速傅里叶变换（FFT）、滤波器组等关键概念及其实际应用。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 掌握数字信号处理的核心技术和连续时间与离散时间采样的高级技能；

✔️ 描述数字信号处理框架及其构建模块的数学表达与应用；

✔️ 在多速率滤波器组处理中开发技能，表达并应用数字信号处理技术；

✔️ 综合应用所学知识解决实际问题。

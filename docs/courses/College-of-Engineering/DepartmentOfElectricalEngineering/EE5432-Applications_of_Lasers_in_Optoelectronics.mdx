# EE5432
## 激光在光电子学中的应用

## **📘 简介**
本课程旨在为学生提供激光理论、设计考量、操作动态及其在光电子学中的应用知识。通过学习激光器在光电子学中的应用，学生将对光电子技术的最新发展产生兴趣。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5432.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5432.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述激光在光电子学中的应用，并匹配激光类型与其应用场景；

✔️ 分析不同激光振荡器的设计并评估其输出性能；

✔️ 讨论激光的动态操作；

✔️ 描述新型激光器件的研究进展；

✔️ 表征光电子学中常用的激光器。

# EE6617
## 检测与估计：通信中的理论与应用

## **📘 简介**
本课程旨在为学生提供检测与估计理论的基本知识及其在通信和信号处理中的应用。学生将通过数学建模和优化工具解决实际问题。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE6617.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE6617.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 探索和讨论检测与估计的一般框架，增强概念理解；

✔️ 通过小组合作，识别并解决检测与估计问题，应用统计信号处理和优化技能；

✔️ 将检测与估计技术应用于通信和信号处理中的实际问题。

# EE5808
## 计算机图形学专题

## **📘 简介**
本课程旨在为学生提供深入的计算机图形学原理、概念和高级技术的批判性理解。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5808.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5808.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 应用 3D 对象表示技术构建图形场景；

✔️ 使用层次结构和坐标变换技术建模和查看连接对象；

✔️ 应用光照、阴影和光栅化技术创建 2D 图像；

✔️ 使用纹理映射和动画技术；

✔️ 应用并评估高级图形技术；

✔️ 使用计算机图形学创建动画或游戏。

# EE5811
## 计算机视觉专题

## **📘 简介**
本课程深入探讨计算机视觉的基本原理、概念及其高级技术。课程涵盖低级图像处理、高级视觉语义理解及其在实际应用中的应用。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5811.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5811.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 应用低级计算机视觉技术分析图像/视频特性；

✔️ 应用高级计算机视觉算法进行图像/视频理解；

✔️ 针对计算机视觉问题应用机器学习算法；

✔️ 使用 Python 或 MATLAB 应用并评估计算机视觉算法。

# EE5806
## 图像处理专题

## **📘 简介**
本课程旨在为学生提供数字图像处理的技术基础，包括图像重建、修复、模式识别和视频分析。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5806.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5806.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述图像变换；

✔️ 描述图像重建和修复；

✔️ 描述图像分割与模式识别；

✔️ 执行视频分析；

✔️ 将计算机算法应用于实际问题。

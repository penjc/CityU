# EE6621
## 计算生理学与神经系统

## **📘 简介**
本课程探索工程原理在生理学和神经科学建模中的应用，通过计算工具更好地理解人体和大脑。学生将学习信号处理、控制系统及电路理论在复杂生物过程建模中的应用，内容涵盖生理学基础、神经系统建模及现代研究方法。

- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE6621.pdf)

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 使用工程原理开发和实现生理学与神经系统模型；

✔️ 应用信号处理技术分析生物电信号，使用机器学习进行模式识别与分类；

✔️ 利用控制理论分析生理系统中的反馈机制；

✔️ 评估并改进复杂生物系统模型，应用于创新性医疗技术。

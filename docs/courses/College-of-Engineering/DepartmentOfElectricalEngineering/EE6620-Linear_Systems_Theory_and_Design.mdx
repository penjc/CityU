# EE6620
## 线性系统理论与设计

## **📘 简介**
本课程为学生提供线性系统分析与设计的基本概念、技术和工具。这些技能适用于系统与控制、信号处理和通信等工程领域。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE6620.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE6620.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别并确定系统特性；

✔️ 描述并应用状态空间建模；

✔️ 实现线性动态系统的解/响应；

✔️ 应用可控性和可观性概念于线性系统；

✔️ 使用状态反馈和状态估计器进行工程设计。

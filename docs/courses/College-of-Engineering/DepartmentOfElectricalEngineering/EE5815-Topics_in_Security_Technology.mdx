# EE5815
## 安全技术专题

## **📘 简介**
本课程介绍计算机安全技术的原理，包括密码学、侧信道攻击、取证及数据、通信和嵌入式系统安全。学生将学习防御技术并设计安全解决方案。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5815.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE5815.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别威胁、漏洞与攻击之间的差异；

✔️ 认识保障系统免受攻击的技术与机制；

✔️ 分析不同嵌入式系统和互联网技术中的安全问题；

✔️ 描述互联网安全的发展趋势及其技术。

# EE5815
## 安全技术专题

## **📘 简介**
本课程介绍计算机安全技术的原理，包括密码学、侧信道攻击、取证及数据、通信和嵌入式系统安全。学生将学习防御技术并设计安全解决方案。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别威胁、漏洞与攻击之间的差异；

✔️ 认识保障系统免受攻击的技术与机制；

✔️ 分析不同嵌入式系统和互联网技术中的安全问题；

✔️ 描述互联网安全的发展趋势及其技术。

## **📊 评估方式**

| **评估项目**          | **权重** | **具体描述**                                                                                       |
|----------------------|---------|---------------------------------------------------------------------------------------------------|
| **📚 测试**            | 30%     | 至少两次测试，评估学生对安全技术的理解。                                                         |
| **📝 作业**           | 10%     | 包括家庭作业与项目报告，评估学生的实际应用能力。                                                  |
| **📚 实验**           | 10%     | 通过实践强化学生对密码学与安全系统的理解。                                                       |
| **📖 期末考试**       | 50%     | 时长为 2 小时的考试，评估学生对课程目标的全面掌握。                                                |

# EE6413
## 网络技术高级专题

## **📘 简介**
本课程旨在为学生提供各种有线和无线网络的原理、协议及性能评估技术的知识。学生将学习如何设计、分析和管理现代网络系统。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE6413.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE6413.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 演示 OSI 网络模型中不同层的功能；

✔️ 描述最新的 MAC 协议及其在 Wi-Fi 和蜂窝网络中的应用；

✔️ 描述最新的路由协议及其在互联网和无线自组网络中的应用；

✔️ 确定实际有线和无线网络的需求和协议设计方法；

✔️ 应用系统管理技术管理使用不同网络技术的网络。

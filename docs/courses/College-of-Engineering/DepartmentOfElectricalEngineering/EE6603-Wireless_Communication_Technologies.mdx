# EE6603
## 无线通信技术

## **📘 简介**
本课程旨在让学生理解无线通信理论的基本原理，尤其是通过探索学习的方式解决实际无线通信网络（如蜂窝网络和 WiFi 网络）中的信号与系统设计问题。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE6603.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/EE6603.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述衰落信道的特性并识别无线信道的统计建模方法；

✔️ 应用分集技术克服衰落问题；

✔️ 解释多址技术及其在实际无线系统中的应用；

✔️ 理解系统需求和评估方法并应用于实际无线系统。

# EE5414
## 嵌入式系统的开发与设计

## **📘 简介**
本课程介绍嵌入式系统的知识和实践经验。课程将研究嵌入式 Linux 内核、设备驱动程序及硬件接口。学生将使用 Android 作为嵌入式系统的开发平台，设计并实现硬件驱动与 Android 应用程序。通过小型项目和实验，学生将实现实时应用并获得实践经验。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述嵌入式 Linux 内核和设备驱动程序；

✔️ 评估 Android 架构及其运行时环境（包括 Dalvik 虚拟机和 Zygote 等）；

✔️ 将 Android 移植到嵌入式设备并配置 Linux 内核与驱动程序；

✔️ 创新性地设计、分析和实现基于 Linux 或 Android 平台的嵌入式系统硬件接口。

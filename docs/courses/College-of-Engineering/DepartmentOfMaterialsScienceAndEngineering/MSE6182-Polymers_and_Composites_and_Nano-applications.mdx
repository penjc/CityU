# MSE6182
## 聚合物与复合材料及其纳米应用

## **📘 简介**
本课程旨在培养学生对聚合物科学与工程的研究技能，覆盖聚合物分子结构、物理和化学性能、复合材料的微观力学理论以及功能性聚合物复合材料的设计与应用。课程还将探讨包括自愈材料、导热材料和仿生复合材料在内的前沿纳米复合材料应用。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述聚合物的分子性质、聚合方法及其性能；

✔️ 应用实验技术表征聚合物及复合材料的行为；

✔️ 创新设计聚合物基体复合材料以实现机械增强和导热性改进；

✔️ 阐述自愈材料和仿生复合材料等功能性聚合物复合材料的特性。

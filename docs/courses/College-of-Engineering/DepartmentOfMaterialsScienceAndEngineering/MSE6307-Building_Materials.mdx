# MSE6307
## 建筑材料

## **📘 简介**
本课程覆盖与结构应用相关的建筑材料的基本结构和性质。学生将学习常见建筑材料的类别、结构和特性，并了解其在结构应用中的实际考虑。

- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/MSE6307.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述结构建筑材料的类型、成分及设计，特别是混凝土和钢筋；

✔️ 解释影响混凝土结构耐久性的因素；

✔️ 选择和应用混凝土及其他建筑材料的多种测试方法；

✔️ 描述幕墙材料的种类及功能；

✔️ 为内部墙体、天花板和隔断选择合适的材料；

✔️ 识别建筑材料领域的最新发现和前沿发展，并针对相关问题发表观点。

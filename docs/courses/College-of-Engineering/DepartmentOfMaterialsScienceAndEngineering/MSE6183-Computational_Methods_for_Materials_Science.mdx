# MSE6183
## 材料科学中的计算方法

## **📘 简介**
本课程介绍科学与工程中使用离散粒子系统和连续场的计算建模基本概念。学生将学习统计采样、模拟、数据分析与可视化技术，使用分子动力学、蒙特卡洛方法及连续介质方法研究材料科学的基础物理现象。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别模拟与建模在材料科学中的重要性；

✔️ 使用模拟与建模工具解决材料科学问题；

✔️ 识别决定模拟质量和可靠性的关键变量；

✔️ 掌握领域最新发展并展示独立问题解决能力；

✔️ 应用基本模拟与建模工具解决材料科学、物理、化学及生命科学领域中的简单问题。

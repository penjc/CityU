# MSE5301
## 材料表征仪器学

## **📘 简介**
本课程介绍材料表征技术的基本理论框架，包括对材料的结构、成分和表面性能的表征。通过该课程，学生将学习材料表征所需的物理、仪器考虑、优点及限制，选择适当的技术和仪器，并获得实验室操作经验。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 解释材料表征技术的理论框架；

✔️ 描述基于各种表征技术的结构、成分、形态和化学信息的定量分析方法；

✔️ 解释检测源（如 X 射线、电子和中子）的仪器设计与生成原理；

✔️ 根据各种仪器的优缺点选择适当的表征技术。

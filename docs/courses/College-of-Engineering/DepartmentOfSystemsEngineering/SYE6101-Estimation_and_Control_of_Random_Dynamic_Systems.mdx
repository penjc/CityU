# SYE6101
## 随机动态系统的估计与控制 (Estimation and Control of Random Dynamic Systems)

## **📘 简介**
本课程旨在教授学生随机动态系统的基本概念和方法，包括工程和经济领域的动态系统的建模、预测、控制和优化。课程特别关注动态编程和强化学习在复杂动态优化问题中的应用。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 建立具有不确定性的动态系统模型；

✔️ 预测系统未来的行为；

✔️ 根据目标选择控制变量；

✔️ 构建不同复杂性的原型；

✔️ 应用数学技术进行动态系统分析。

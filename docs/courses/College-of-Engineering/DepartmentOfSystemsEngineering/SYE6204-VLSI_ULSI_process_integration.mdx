# SYE6204
## VLSI/ULSI 工艺集成 (VLSI/ULSI Process Integration)

## **📘 简介**
本课程介绍超大规模集成 (VLSI) 和超超大规模集成 (ULSI) 的基本知识，涵盖数百万或数十亿个 MOS 晶体管集成到单片芯片中的过程。学生将学习 MOS 晶体管理论、IC 制造工艺、后端设计基础、计算机辅助设计工具及其方法。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202425/course/SYE6204.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/SYE6204.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 掌握 MOS 晶体管的基础知识；

✔️ 理解微加工工艺的基本概念；

✔️ 熟悉 VLSI 设计基础知识；

✔️ 评估芯片的测试、可靠性、功耗和性能。

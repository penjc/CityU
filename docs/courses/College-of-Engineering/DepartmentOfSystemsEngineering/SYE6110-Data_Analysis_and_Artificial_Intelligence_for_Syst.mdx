# SYE6110
## 系统工程中的数据分析与人工智能 (Data Analysis and Artificial Intelligence for Systems Engineering)

## **📘 简介**
本课程涵盖数据驱动的人工智能基础知识，特别是基于机器学习的方法，包括分类和回归、无监督聚类、深度学习等。课程提供实践机会，使用 Python 工具开发统计、数据分析和人工智能模型，并聚焦智能制造案例。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 阐述数据分析问题的关键问题并将其转化为机器学习或人工智能模型；

✔️ 根据问题选择合适的数据分析或机器学习技术；

✔️ 运用相关技术对数据进行有效分析并评估结果；

✔️ 使用 Python 模块解决与制造相关的问题。

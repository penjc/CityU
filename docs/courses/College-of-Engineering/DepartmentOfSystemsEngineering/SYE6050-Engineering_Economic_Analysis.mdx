# SYE6050
## 工程经济分析 (Engineering Economic Analysis)

## **📘 简介**
本课程旨在帮助学生掌握经济和财务分析的基本原理，并将其应用于工业和工程管理中的财务决策。课程内容包括运营和资本预算、财务报表分析、资本成本与租赁分析，以及项目风险与投资分析。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/SYE6050.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/SYE6050.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 解释货币的社会和商业价值；

✔️ 阐述财务管理的基本原理，并讨论财务报表的内容；

✔️ 从工程经济学的角度分析财务报表并制定投资策略；

✔️ 描述项目风险的性质，并对特定项目的风险进行评估；

✔️ 从财务和非财务的角度评估决策选项。

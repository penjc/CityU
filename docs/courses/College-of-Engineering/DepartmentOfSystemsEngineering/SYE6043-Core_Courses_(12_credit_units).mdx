# SYE6043
## 质量与可靠性工程 (Quality and Reliability Engineering)

## **📘 简介**
本课程旨在为学生提供评估和改进产品或过程质量和可靠性的基本方法和技术，重点介绍统计质量控制和现代可靠性工程工具的原理及其实际应用。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202425/course/SYE6043.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/SYE6043.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 掌握质量改进的基本技术和统计与概率基础知识；

✔️ 应用控制图分析和改进过程质量；

✔️ 设计简单的抽样计划及其操作特性 (OC) 曲线分析；

✔️ 获取系统可靠性计算和模型计算的基础知识；

✔️ 理解实验设计的基本知识，特别是因子设计矩阵和田口损失函数。

# SYE6012
## 技术创新与创业 (Technological Innovation and Entrepreneurship)

## **📘 简介**
本课程旨在培养学生开发创新技术产品的理解能力，以及有效管理开发项目的技能和技巧。学生将学习产品和流程创新的基本原理，并从企业家视角开展创新项目。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/SYE6012.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/SYE6012.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 通过小组讨论和头脑风暴开发新想法，结合技术推动和市场拉动来源；

✔️ 理解产品和流程创新的基本过程与原理，分辨创造性思维和批判性思维的不同；

✔️ 从创业角度而非工程角度整合市场、财务及团队管理问题，完成创新项目；

✔️ 识别日常生活和工作中的创新实例，并从中获得启发；

✔️ 结合工程和管理理论，制定完整的创新与创业项目计划，包括创意设计、专利搜索、市场及财务计划。

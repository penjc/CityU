# CA5301
## 建筑控制、合同管理与伦理专业实践

## **📘 简介**
本课程旨在为学生提供对关键城市规划与建筑法规、建筑合同和专业伦理的全面理解。学生需要整合并应用与建筑项目相关的法定与非法定控制措施。课程介绍了各种建筑合同的形式及招标程序，并旨在提高学生对专业历史与伦理问题的认识。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 了解建筑实践作为一项职业的历史及其对社会的责任；

✔️ 提高对建筑实践中专业伦理的认知；

✔️ 解释建筑控制的原理，以及关键建筑法规的应用；

✔️ 描述建筑项目中各种合同形式及其招标程序，并理解其应用。

# CA5251
## 可持续建筑发展

## **📘 简介**
本课程旨在为学生提供可持续建筑发展的知识，涵盖政府政策与法规、绿色建筑认证体系、绿色建筑设计、可持续建筑施工技术与材料、建筑运营与改造、建筑拆除与材料回收等建筑生命周期中的各个方面。

- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/CA5251.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 理解可持续建筑发展的法定规定和自愿认证体系；

✔️ 解释并应用被动和主动建筑设计概念；

✔️ 了解并描述可持续建筑施工技术和建筑材料；

✔️ 理解可持续建筑运营和改造的要求；

✔️ 理解建筑拆除中的材料回收技术和要求。

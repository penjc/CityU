# CA5018
## 建筑环境建模与计算技术

## **📘 简介**
本课程提供了建筑环境建模和计算技术的理论和计算机实现方法。通过本课程，学生将了解如何应用计算技术来模拟建筑环境中的问题，掌握使用商业软件包的能力，以解决工程研究中的实际问题。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202425/course/CA5018.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/CA5018.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 理解并应用建模技术和计算机软件包，解决与工程研究相关的问题；

✔️ 探索不同建模技术的性质，并能够应用于实际工程问题；

✔️ 理解经验方法的局限性，并能够使用仿真模型解决工程研究中的问题。

# CA5152
## 高级建筑设计工作室：保护

## **📘 简介**
本课程提供了建筑设计中保护遗产建筑和历史场所的全面学习，重点探讨如何在现代建筑设计中实施历史建筑的适应性再利用和城市设计。通过实践和理论结合，学生将能为历史遗产保护提供创新的设计方案，推动可持续发展的城市设计。

🔗 **相关链接**

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别历史遗产保护项目中的背景问题，结合建筑和可持续发展的新兴话题进行分析；

✔️ 提出新的建筑或城市保护策略，通过设计概念和方案提出创新的保护方法；

✔️ 设计针对当代问题的保护项目，解决历史建筑保护中的挑战；

✔️ 准备专业的设计方案并进行展示和文档编制。

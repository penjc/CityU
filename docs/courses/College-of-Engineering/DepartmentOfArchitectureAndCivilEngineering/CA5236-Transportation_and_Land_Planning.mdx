# CA5236
## 交通与土地规划

## **📘 简介**
本课程旨在帮助学生理解交通在城市和区域发展中的作用，探讨城市人交通系统的特点，分析和预测城市交通需求的方法，并讨论交通系统管理与资本改进的规划。课程还涉及大规模公共交通系统、高速列车以及它们与区域规划的关系。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 探讨土地使用规划与交通之间的关系；

✔️ 应用交通需求、可达性、经济、环境和社会影响的分析方法；

✔️ 理解城市公共交通、高速铁路和机场系统在区域规划中的作用；

✔️ 理解交通需求管理、以交通为导向的发展（TOD）和可持续基础设施融资的概念。

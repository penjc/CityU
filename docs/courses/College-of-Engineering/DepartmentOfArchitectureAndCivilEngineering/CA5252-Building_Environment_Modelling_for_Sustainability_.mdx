# CA5252
## 建筑环境建模与可持续性分析

## **📘 简介**
本课程旨在为学生提供流行的建筑能源建模工具概述；理解建筑能源系统建模的基本原理；并能在系统设计和设施管理中应用。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202425/course/CA5252.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/CA5252.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述常见建筑能源建模工具的功能和特点；

✔️ 讨论建模典型建筑能源系统（如HVAC和照明）的原理、机制、假设和局限性；

✔️ 讨论建模先进建筑能源系统（如三联供和可再生能源系统）的原理、机制、假设和局限性；

✔️ 将建筑能源建模应用于系统设计、能源管理和重新调试。

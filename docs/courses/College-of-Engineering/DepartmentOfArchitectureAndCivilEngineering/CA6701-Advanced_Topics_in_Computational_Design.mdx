# CA6701
## 计算设计高级专题

## **📘 简介**
本课程提供建筑领域计算设计的高级知识。学生将学习通过编程语言研究计算与参数化设计，探索计算设计中的理论、方法论和技术，重点关注通过编程和脚本创建新型建筑形式。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202425/course/CA6701.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/CA6701.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别计算设计和编程语言的理论、方法论和技术基础；

✔️ 研究计算设计及其应用；

✔️ 使用参数化设计、脚本和编程创建创新建筑形式；

✔️ 提出新的、先进的建筑形式创建逻辑。

# CA6241
## 地理数据管理与规划分析

## **📘 简介**
本课程旨在帮助学生理解地理信息科学（GIS）的基本原则，及其在地理空间、环境和土地使用规划应用中的智能使用。学生将学习如何应用GIS解决相关的规划问题，并掌握分析区域作为经济、社会和空间系统的技巧和数据来源。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 发现地理信息科学在土地使用规划、城市需求和资产管理等方面的应用；

✔️ 应用规划响应所需的分析方法；

✔️ 探索GIS技术在实际环境中的应用潜力。

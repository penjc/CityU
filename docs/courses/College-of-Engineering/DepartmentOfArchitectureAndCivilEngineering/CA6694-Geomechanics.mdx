# CA6694
## 岩土力学

## **📘 简介**
本课程介绍岩土力学（土壤与岩石力学）和岩土工程学的基本概念和理论。课程将介绍分析和经验方法，帮助学生解决岩土设计问题。旨在培养学生对岩土结构及土壤行为的好奇心和独立探索的能力。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202425/course/CA6694.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/CA6694.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 解释岩土力学的理论和分析/经验方法的重要性；

✔️ 体验土壤力学实验方法，重点研究国际最新进展；

✔️ 预测地面和地质系统在不同载荷条件下的行为；

✔️ 使用土壤模型表征地面行为；

✔️ 在遇到新土壤时理解探索性思维的重要性。

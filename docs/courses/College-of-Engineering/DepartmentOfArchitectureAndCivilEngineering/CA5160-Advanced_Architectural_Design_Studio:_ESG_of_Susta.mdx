# CA5160
## 高级建筑设计工作室：住房与社区

## **📘 简介**
本课程旨在提供一个综合性的设计工作室环境，帮助学生发展住房设计和社区规划的高级技能。学生将通过项目实践，探索如何设计符合社会可持续性的住房，创造能够促进社区互动和归属感的环境。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别住房设计中的背景问题，考虑建筑和社会可持续性的新兴话题；

✔️ 制定创新的设计概念和方案，提出适应现代社会需求的住房设计策略；

✔️ 完成一个住房设计项目，解决当代城市和社区问题；

✔️ 准备专业的设计提案和展示文档。

# CA5564
## 智能建筑的传感与数据分析

## **📘 简介**
本课程提供关于智能建筑中传感技术和数据分析的基础知识。课程涵盖与传感器、传感技术、物联网、数据分析、机器学习等相关的主题，为学生提供实施现代电子和信息技术所需的基础技术背景。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202425/course/CA5564.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/CA5564.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 了解智能建筑中传感和传感网络技术的基本原理；

✔️ 理解智能建筑中的数据分析应用；

✔️ 理解传感和数据分析理论的基本原理和背景。

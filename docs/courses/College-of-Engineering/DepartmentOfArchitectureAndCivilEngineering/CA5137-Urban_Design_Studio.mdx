# CA5137
## 城市设计工作室

## **📘 简介**
本课程提供学生关于香港及其他地区城市设计的全面了解，强调本地城市设计和规划中的实际问题。课程通过跨学科的方法，帮助学生运用最新的城市设计策略，进行区域规划和城市设计，提升学生的分析能力和创造性思维。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202425/course/CA5137.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/CA5137.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别香港城市设计中的主要问题，并结合最新的设计主题进行分析；

✔️ 将不同学科的方法整合到城市设计中，提出创新的设计概念和方案；

✔️ 设计一个基于香港新开发区域的城市设计项目；

✔️ 准备专业的展示和文档，提出新区域的城市设计方案。

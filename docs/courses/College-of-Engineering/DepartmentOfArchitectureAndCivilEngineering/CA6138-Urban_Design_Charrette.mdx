# CA6138
## 城市设计工作坊

## **📘 简介**
本课程旨在提供学生应用最先进的城市设计方法，聚焦于区域、国家和国际城市设计及规划问题。通过设计驱动的探索，学生将开发新设计策略以解决多样的城市问题，并使用计算工具分析情境与可视化策略。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别全球各城市面临的城市问题并学习先进的设计方法；

✔️ 通过采用新的设计方法，制定设计概念和方案；

✔️ 为选定的城市设计一个完整的城市设计项目，并开发新的设计逻辑和方案；

✔️ 准备专业的城市设计提案，并用合适的方式呈现。

# MNE6001
## CAD/CAM 集成

## **📘 简介**
本课程旨在帮助学生掌握计算机辅助设计与制造（CAD/CAM）的技术基础，并学习如何应用相关技术解决具有几何复杂性的设计与制造问题。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/MNE6001.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/MNE6001.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述几何实体表示的数学基础，包括参数曲线和自由曲面；

✔️ 阐述实体建模和其他高级表示方法的基本理论和算法；

✔️ 解释 3 轴和多轴加工的 CNC 刀具路径计算技术；

✔️ 应用相关技术设计简单的 CAD/CAM 算法；

✔️ 解决涉及几何的设计/制造问题并转化为算法问题。

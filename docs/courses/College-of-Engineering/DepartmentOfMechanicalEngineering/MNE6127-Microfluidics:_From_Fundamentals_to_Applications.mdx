# MNE6127
## 微流控：从基础到应用

## **📘 简介**
微流控技术涉及在微米尺度上操控和处理少量液体，现已成为一个多学科交叉的研究领域。课程内容包括微流控系统的基础概念、制造方法、分类及其在工程、化学、生物和医学中的实际应用，旨在培养学生分析和设计微流控系统的能力。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/MNE6127.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/MNE6127.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述微流控技术的基础知识与应用；

✔️ 解释微尺度流体动力学并解决相关问题；

✔️ 识别实际产品中的微流控系统并通过小型项目展示分析能力；

✔️ 应用相关知识设计研究和开发用的微系统。

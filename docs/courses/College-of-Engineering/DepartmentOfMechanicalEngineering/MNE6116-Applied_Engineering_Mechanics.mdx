# MNE6116
## 应用工程力学

## **📘 简介**
本课程教授高级工程力学知识，并将其应用于研究与开发。内容涵盖弹性与塑性、粘弹性、失效理论、断裂、疲劳及有限元理论，并介绍其在变形材料与软体机器人中的应用。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述工程力学的基本概念及其在机械应用中的影响；

✔️ 识别工程力学相关问题，并运用力学理论计算与分析；

✔️ 针对力学问题进行文献调研并完成小型项目。

# MNE6130
## 现代机器人学

## **📘 简介**
机器人技术是一个快速发展的领域，广泛应用于家居、服务和工业环境。本课程从研究生水平深入探讨机器人技术，内容涵盖机器人系统的分析、建模与控制的基本概念与方法，包括运动学、逆运动学、动力学、运动控制与规划、人机交互、机器人传感与视觉、机器人编程与应用。学生将在课程中参与编程项目，以巩固理论知识。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述机器人系统的基本概念、分类及实际应用；

✔️ 阐释机器人运动学、动力学、运动规划与控制、传感与编程的原理；

✔️ 识别家用、服务与工业应用中合适的机器人需求；

✔️ 应用所学概念、原理与方法设计并开发机器人以满足不同需求。

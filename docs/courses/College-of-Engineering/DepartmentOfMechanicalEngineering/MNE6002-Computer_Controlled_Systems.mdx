# MNE6002
## 计算机控制系统

## **📘 简介**
本课程旨在培养学生对自动化系统实时控制的深入理解，特别是通过数字计算机实现控制的应用。学生将学习离散域分析与设计的数学技术，并增强其分析、设计和实施控制系统的技能。

- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/MNE6002.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 阐述数字控制的基本原理及其实现方法；

✔️ 使用 z-变换分析离散时间系统；

✔️ 使用 z 平面和频域方法设计离散时间控制系统；

✔️ 为离散时间系统设计基于状态空间的控制器；

✔️ 应用高级控制方法实现动态系统（如机器人）的计算机控制。

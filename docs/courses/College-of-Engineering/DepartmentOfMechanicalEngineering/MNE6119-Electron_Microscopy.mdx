# MNE6119
## 电子显微镜学

## **📘 简介**
本课程聚焦于现代电子显微镜技术的理论与应用，包括扫描电子显微镜（SEM）、透射电子显微镜（TEM）和 X 射线能谱（EDS）。学生将学习基本电子光学、电子束与样品的相互作用、电子衍射、高级成像技术以及定性和定量 X 射线微分析。此外，课程强调实践经验，包括样品制备技术和显微镜操作。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述电子显微镜及光谱技术的理论与应用；

✔️ 阐释 SEM/TEM/EDS 的基本原理和硬件结构；

✔️ 应用 SEM/TEM/EDS 技术进行成像、衍射及光谱实验和数据分析。

# MNE6129
## 可持续工程实践

## **📘 简介**
本课程旨在让学生掌握可持续工程技术的实验技能和实际应用能力。课程内容以实验为主，涵盖数据采集、实验数据分析以及良好的报告撰写实践。通过该课程，学生将学会将理论知识与实验实践相结合。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 应用热力学与传热理论分析能效系统，例如制冷循环与换热器设计，并进行数据采集与分析；

✔️ 探讨可再生能源转换系统的理论，如光伏太阳能电池、燃料电池和生物燃料转换，并进行实验数据分析与报告；

✔️ 运用环境治理技术的理论进行空气与废水净化实验，并进行数据分析；

✔️ 通过口头演讲有效传达实验结果。

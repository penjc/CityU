# MNE6115
## 仿生机器人

## **📘 简介**
本课程通过研究自然界中的生物机制，指导学生设计并实现仿生机器人系统。课程内容包括动力学、尺度物理学、多代理系统以及分散系统在机器人学中的应用。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202425/course/MNE6115.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/MNE6115.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 比较并描述工程系统、仿生系统及仿生机器人系统；

✔️ 推导飞行、行走等生物和机器人运动的动力学规律；

✔️ 探讨多代理和分散系统的基本原理及其在机器人学中的应用；

✔️ 分析生物系统的潜在机制，并将其应用于工程设计。

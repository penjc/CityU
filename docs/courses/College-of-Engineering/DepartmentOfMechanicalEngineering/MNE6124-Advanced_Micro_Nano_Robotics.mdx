# MNE6124
## 微/纳米机器人学

## **📘 简介**
微/纳米机器人学是一门跨学科课程，涉及微制造、机器人学、医学和材料科学。课程涵盖微型机器人及微/纳米操控系统的设计、建模、制造与控制的基本原理，以及相关案例研究、挑战、未来趋势及应用。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 理解微/纳米机器人和自动化的尺度因子挑战；

✔️ 回顾微/纳米机器人技术和 MEMS 设计原则；

✔️ 应用适当理论与制造技术实现自动化操控；

✔️ 解决生物医学和制药等领域的实际问题。

# MNE6128
## 工程中的高级机器学习与量子计算

## **📘 简介**
本课程介绍现代计算领域的两个前沿主题——机器学习和量子计算。课程内容涵盖机器学习的基础与应用，以及量子计算的硬件设计和基础算法。通过使用 Matlab 和 Qiskit，学生将学习如何应用这些技术解决机械工程中的实际问题。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/MNE6128.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/MNE6128.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 确定可以通过机器学习或量子计算解决的工程问题；

✔️ 描述机器学习和量子计算的原理及核心模型；

✔️ 应用这些技术于具体工程问题；

✔️ 分析和评估模型的准确性及适用性；

✔️ 使用 Matlab 和 Qiskit 实现基本模型。

# MNE6007
## 高级自动化技术

## **📘 简介**
本课程为学生提供对自动化领域重要技术基本原理的理解。课程内容涵盖机器人技术和机器视觉的理论与应用，为学生解决工程问题并在自动化和其他应用中设计合适的工程解决方案奠定基础。

- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/MNE6007.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述机器人和机器视觉的基本理论；

✔️ 解决基本视觉问题；

✔️ 分析视觉系统的原理；

✔️ 设计用于自动化的机器人系统；

✔️ 应用机器人和传感系统解决工程实际问题。

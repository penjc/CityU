# MS5223
## 项目管理

## **📘 简介**
本课程为学生提供项目管理的基本概念的全面介绍，重点是项目管理中的关键权衡。学生将掌握旨在帮助项目经理在各个行业中使用的基本工具和方法。课程内容涵盖了项目管理概述、详细的项目规划、计算机化项目管理系统的使用、项目图表、定价与估算、成本控制策略等。强调项目权衡与风险分析、并行工程、以及将全面质量管理（TQM）技术与有效项目规划结合起来。通过理论课程、实际作业、现实项目和案例研究的结合，学生将培养应用项目管理概念和工具的技能，帮助他们处理复杂的项目并推动成功的结果。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 基于战略目标、财务影响和资源分配做出项目组合的明智决策；

✔️ 创建全面的项目计划，确保及时且成本效益高地完成项目，并展示项目监控、风险管理和沟通管理方面的专业能力；

✔️ 认识到项目管理办公室在项目成功中的关键作用，并理解平衡项目管理的技术与社会文化方面的需求；

✔️ 熟练使用当代项目管理软件，如 Microsoft Project，有效管理项目各个阶段，并运用电子表格模型作为决策支持工具。

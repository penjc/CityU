# MS6711
## 数据挖掘

## **📘 简介**
本课程介绍了与商业应用相关的流行和实用的数据挖掘与机器学习算法。学生将使用 Python 编程语言进行数据分析。在课程结束时，学生将具备数据挖掘领域的核心基础知识，为从事数据分析相关职业打下坚实的基础。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/MS6711.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/MS6711.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 掌握最先进的机器学习和数据挖掘算法；

✔️ 定义并制定现实世界的数据挖掘问题，准备数据并执行算法，解读结果；

✔️ 使用 Python 分析现实世界的数据。

---

## **📊 评估方式**

| **评估项目**          | **权重** | **具体描述**                                                                                       |
|----------------------|---------|---------------------------------------------------------------------------------------------------|
| **📄 作业**           | 10%     | 学生通过作业解决实际数据挖掘问题，并应用相关算法。                                                 |
| **📊 项目**           | 50%     | 学生将接收一个大型数据集，并根据预定的数据挖掘目标提取有价值的洞察。                             |
| **📝 期末考试**       | 40%     | 期末考试评估学生对数据挖掘算法的理解与应用能力。                                                   |

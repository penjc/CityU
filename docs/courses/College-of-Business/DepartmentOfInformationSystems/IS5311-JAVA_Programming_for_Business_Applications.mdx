# IS5311
## 商业应用的JAVA编程

## **📘 简介**
本课程旨在介绍解决商业问题所需的编程概念与技能，重点是面向对象编程（OOP）在商业信息系统构建中的应用。完成本课程后，学生应能使用Java解决基本问题，并开发简单的商业应用程序。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202425/course/IS5311.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/IS5311.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述商业软件构建的基础；

✔️ 设计并开发适用于商业应用程序的控制结构；

✔️ 设计并开发适用于商业软件构建的模块化结构；

✔️ 设计并开发适用于商业软件构建的简单数据结构和算法。

## **📊 评估方式**

| **评估项目**          | **权重** | **具体描述**                                                                                       |
|----------------------|---------|---------------------------------------------------------------------------------------------------|
| **📄 课堂与实验作业** | 10%     | 学生将在课堂和实验期间完成相关编程任务，展示对控制结构的理解。                                    |
| **📄 项目**           | 40%     | 学生需设计并开发一个解决商业问题的程序模块，展示其应用Java的能力。                              |
| **📄 实验测试**       | 50%     | 个人实验测试评估学生在编程技能和应用控制结构方面的掌握，测试时长为2小时。                        |

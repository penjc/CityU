# COM5508
## 媒体数据分析

## **📘 简介**
本课程旨在教授传播与新媒体学生使用计算社会科学方法、工具和算法对社交媒体数据进行分析与可视化的技能。课程重点包括构建、验证及应用预测模型分析社交媒体用户行为，并使用多种方法探索、分析和可视化大数据。同时，还会讨论政策与伦理问题，如隐私保护、数据完整性与开放访问。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 理解分析与可视化社交媒体数据的原则与程序，并具备自主学习能力；

✔️ 掌握数据分析的基本方法与技术，评估其优缺点；

✔️ 系统分析用户生成内容的特征与模式；

✔️ 评估数据分析与可视化中的伦理与社会责任；

✔️ 展现批判性思维能力，制定并实施社交媒体数据分析计划。

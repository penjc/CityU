# COM5506
## 传播的社会网络分析

## **📘 简介**
本课程旨在培养学生“网络化视角”，将传播过程中的实体（如传播者、受众、媒体组织等）视为通过直接或间接、正式或非正式、显性或隐性的方式相互连接的“网络节点”。课程内容涵盖社会网络分析的基本理论与研究方法，并探索其在传播中的多种应用，包括在线社交网络的人际互动、信息扩散、跨国媒体内容流动、口碑营销和健康传播模型等。学生将学习如何设计社会网络分析研究，收集、整合、分析和可视化社会网络数据，并将网络化视角应用于解决实际传播问题。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述并解释社会网络和复杂网络的基本理论概念和研究方法；

✔️ 收集、分析、解释并可视化社会网络数据以解决实际问题；

✔️ 在社会、商业或工程环境中应用理论视角和方法论；

✔️ 以专业质量和风格呈现研究成果和案例研究。

# COM5108
## 新媒体的心理处理

## **📘 简介**
本课程探索媒体心理学和新媒体研究领域的前沿研究，包括人机交互 (HCI) 和计算机媒介传播 (CMC) 的相关课题。涉及的界面和技术包括互联网、虚拟现实、移动媒体以及计算机和视频游戏等，通过案例研究和理论应用分析新媒体技术对社会和心理的影响。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 讨论新媒体界面设计和可用性中的主要心理学问题；

✔️ 评估与新媒体技术心理效应相关的理论和研究；

✔️ 从实证科学的角度分析新媒体技术的社会和心理影响；

✔️ 将媒体心理学的主要理论应用于实际新媒体设计和使用问题。

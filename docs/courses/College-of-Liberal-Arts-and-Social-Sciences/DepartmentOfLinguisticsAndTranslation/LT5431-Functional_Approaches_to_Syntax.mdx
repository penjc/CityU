# LT5431
## 功能语言学视角下的句法研究

## **📘 简介**
本课程旨在全面介绍当前的句法理论，特别强调功能语言学对句法的研究方法。课程探讨形式和功能范式的异同，包括理论前提、研究关注点、数据收集方法、分析方法和解释原则。通过逐步深入的课程设计，学生将熟悉功能理论及其对语言作为交流编码工具的解释机制。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 阅读和讨论学术文献并参与小组讨论；

✔️ 展示批判性思维、团队合作和演讲技能；

✔️ 阐释形式和功能句法分析的理论构建；

✔️ 分析与名词、动词、主题与主语等相关的语法功能映射关系；

✔️ 应用课堂所学的理论分析语言数据，完成最终项目。

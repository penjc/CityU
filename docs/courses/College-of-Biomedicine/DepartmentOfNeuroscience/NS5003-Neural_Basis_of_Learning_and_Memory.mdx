# NS5003
## 学习与记忆的神经基础

## **📘 简介**
本课程为学生提供学习和记忆的神经科学基础，涵盖经典的脊椎动物和无脊椎动物动物模型，现代的学习和记忆研究方法，突触可塑性和强化学习的细胞机制，以及神经调节。课程将探讨学习与记忆的细胞和分子机制，以及神经退行性病变和人工神经网络的应用。学生将有机会学习如何进行实验设计并分析学习和记忆的研究成果。

## **🎯 学习目标**

完成课程后，学生将能够：


✔️ 描述学习和记忆的现象学、历史和研究领域；

✔️ 解释当前的学习和记忆调节理论；

✔️ 理解突触可塑性和细胞机制对学习和记忆的关系；

✔️ 介绍动物模型、实验设计及创新治疗方法；

✔️ 解释人工神经网络及其在机器学习中的潜在应用。

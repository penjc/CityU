# NS5004
## 分子与细胞神经科学

## **📘 简介**
本课程介绍神经元功能的分子和细胞机制。学生将学习细胞内运输、突触的形成和可塑性，以及神经退行性疾病和神经发育障碍中分子和细胞的异常。课程内容还包括细胞骨架、神经营养因子、RNA干扰和信号转导机制，帮助学生深入理解神经生物学领域的最新前沿和技术。

## **🎯 学习目标**

完成课程后，学生将能够：


✔️ 描述控制神经功能的主要分子途径和细胞分区；

✔️ 发展批判性思维，评估分子和细胞神经科学的新发现；

✔️ 应用神经元的分子组成知识，解释其在特定疾病中的异常如何影响大脑功能；

✔️ 了解分子和细胞神经科学研究的主要实验方法。

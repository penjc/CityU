# NS5002
## 神经生物学与疾病

## **📘 简介**
本课程旨在让学生深入理解神经系统疾病的病理生理学及其治疗方法。课程重点讨论神经退行性疾病（如帕金森症）、急性伤害（如中风）、神经发育障碍（如自闭症）和神经精神障碍（如精神分裂症）。学生将学习疾病的基础知识、分子机制、现有治疗方法及正在进行的转化研究。

## **🎯 学习目标**

完成课程后，学生将能够：


✔️ 了解中枢神经系统的解剖和生理学；

✔️ 识别神经系统主要的创伤性、神经退行性、神经发育性疾病的症状和基本原理；

✔️ 掌握用于研究神经系统疾病的实验模型；

✔️ 分析和解读神经疾病的前临床研究；

✔️ 理解当前神经系统疾病的治疗原则和方法。

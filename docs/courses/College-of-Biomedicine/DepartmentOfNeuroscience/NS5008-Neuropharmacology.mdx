# NS5008
## 神经药理学

## **📘 简介**
神经药理学是研究药物对神经系统的作用机制及其效应的学科。本课程将介绍神经递质系统的基本原理，重点讨论神经递质受体（如GABA受体）及其在神经系统中的作用，以及药物如何通过与这些受体结合来调节大脑功能。课程还将深入探讨神经药物在神经退行性疾病、神经血管疾病、癫痫和精神障碍等常见神经疾病中的应用。

## **🎯 学习目标**

完成课程后，学生将能够：


✔️ 理解神经药理学的基本原理，掌握药物对神经系统的作用；

✔️ 掌握神经递质系统在生理和病理条件下的功能；

✔️ 了解神经疾病的病理机制及药物治疗在缓解症状中的作用；

✔️ 运用神经药理学原理辅助开发药物，调节大脑中受损的突触和神经信号。

# NS5006
## 认知与行为神经科学

## **📘 简介**
本课程讲解涉及感知、信息存储以及如何用这些信息指导人类行为的心智过程。课程内容包括神经活动和感知、物体识别、语言、注意力、基本行为如运动控制、动机（如食欲驱动）、决策和反应产生，此外还包括工作记忆、情绪和意识等高级认知功能。课程不仅提供了普遍的理论概念，还通过计算建模、遗传学操作、神经影像学和脑电图等实践课程帮助学生加深理解。特别关注人类健康和疾病（如神经发育性和神经退行性疾病）对这些过程的影响。

## **🎯 学习目标**

完成课程后，学生将能够：


✔️ 理解与所有主要认知功能相关的生物学概念，以及它们如何影响人类行为；

✔️ 理解调控人类认知功能和行为的神经回路的机制、发展及可能的干扰；

✔️ 理解通常伴随认知功能缺陷和/或行为异常的神经疾病的病理和发病机制。

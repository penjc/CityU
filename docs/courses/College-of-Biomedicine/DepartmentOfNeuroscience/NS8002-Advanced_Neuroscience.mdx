# NS8002
## 高级神经科学

## **📘 简介**
本课程提供神经科学的基础和进阶知识，涵盖从细胞神经生理学到系统神经科学的广泛主题。课程将讨论神经解剖学、突触传递、电生理记录、以及学习、记忆、疼痛和愉悦等基本神经过程，旨在为学生提供深入的神经生物学基础。

## **🎯 学习目标**

完成课程后，学生将能够：


✔️ 描述中枢和外周神经系统的解剖结构和功能；

✔️ 解释动作电位、膜电位、通道、受体、神经递质释放等基本神经生理学概念；

✔️ 了解神经系统如何生成学习、记忆、意识以及疼痛和愉悦等神经生物学过程。

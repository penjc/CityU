# NS5005
## 感觉与运动神经科学

## **📘 简介**
本课程旨在为学生提供神经科学的坚实基础，着重研究神经系统如何通过感觉过程收集环境信息，并如何利用这些信息控制身体的自愿运动。课程重点讲解哺乳动物神经系统，但也简要涉及低等脊椎动物和无脊椎动物的例子。课程将全面考察主要感觉系统（视觉、听觉、触觉、嗅觉、味觉、平衡）的结构和功能，并介绍运动控制的关键环节，从骨骼肌的运动单元到皮层、小脑和基底神经节在运动规划和选择中的作用。

## **🎯 学习目标**

完成课程后，学生将能够：


✔️ 理解视觉、听觉、嗅觉、味觉、触觉和平衡的神经基础；

✔️ 使用视觉系统的知识解释视觉错觉；

✔️ 理解神经系统如何控制肌肉及其对运动的调节作用；

✔️ 描述从骨骼肌运动单元到大脑皮层、小脑及基底神经节在运动选择中的作用。

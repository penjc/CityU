# BMS8111
## 免疫学与传染病

## **📘 简介**
本课程深入探讨先天免疫和适应性免疫的高级知识，以及病原体入侵、复制和在人类和动物种群中扩散的机制。课程将覆盖免疫系统中的信号通路，探讨感染和自身免疫功能障碍。学生还将学习多种病原体，包括病毒、细菌、原生动物和蠕虫病原体，以及它们在不同宿主中的急性和慢性感染机制。

## **🎯 学习目标**

完成课程后，学生将能够：


✔️ 掌握先天免疫和适应性免疫的基础知识；

✔️ 理解感染和自身免疫功能障碍中的信号通路；

✔️ 学习并了解多种病原体及其在宿主中建立急性和慢性感染的机制；

✔️ 掌握特定疾病的感染性病原体基础知识。

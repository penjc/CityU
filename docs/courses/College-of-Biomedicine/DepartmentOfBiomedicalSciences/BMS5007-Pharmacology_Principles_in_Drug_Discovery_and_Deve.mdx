# BMS5007
## 药理学在药物发现与开发中的基本原理

## **📘 简介**
本课程旨在为学生提供药理学在药物发现与开发中的基本原理。涵盖的主题包括（1）蛋白质结构、受体理论、酶动力学和细胞信号通路；（2）药物的吸收、分布、代谢、排泄和毒性（DMPK）；（3）药物发现的关键阶段，如靶点选择和验证、早期先导化合物的筛选及优化、适用于临床前开发的化合物；（4）从药物发现到临床开发的关键阶段及其挑战，医疗和经济因素如何影响药物发现进程；（5）药物发现中的启用技术：高通量筛选、基于结构的药物设计、分子建模、药物筛选和转化医学。

## **🎯 学习目标**

完成课程后，学生将能够：


✔️ 理解药物发现与开发中的药理学基本概念；

✔️ 解释药物的靶点选择、先导化合物筛选、药物优化等过程；

✔️ 讨论药物发现中的技术与方法，了解药物开发的关键挑战；

✔️ 应用药理学知识评估药物的临床前开发阶段。

# BMS8107
## 癌症生物学与精准医学

## **📘 简介**
本课程介绍人类癌症的遗传基础，包括突变机制、癌基因激活、肿瘤抑制基因失活、癌基因与肿瘤抑制基因在细胞周期和凋亡调控中的作用。课程还将重点讨论现代癌症治疗方法的原理和应用，并涉及癌症干细胞及基于癌症干细胞的治疗方法。

## **🎯 学习目标**

完成课程后，学生将能够：


✔️ 描述癌症生物学中的核心主题；

✔️ 识别癌症生物学的细胞基础和分子机制；

✔️ 结合人类癌症的遗传基础；

✔️ 设计概念图，展示人类癌症的遗传基础。

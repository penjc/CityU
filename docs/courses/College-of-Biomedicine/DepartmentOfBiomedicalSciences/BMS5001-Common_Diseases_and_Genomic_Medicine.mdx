# BMS5001
## 常见疾病与基因组医学

## **📘 简介**
本课程旨在让学生深入了解几种常见疾病，包括心血管疾病、内分泌疾病、神经退行性疾病、消化系统疾病、风湿性疾病和癌症，重点介绍基于基因组学医学的前沿技术与治疗方法。学生将学习如何利用现代基因组技术对这些疾病进行诊断、治疗与预防。课程强调香港及亚洲公共卫生中的相关疾病管理与最新进展。

## **🎯 学习目标**

完成课程后，学生将能够：


✔️ 描述心血管、内分泌、神经退行性、消化系统、风湿性及癌症等常见疾病的流行病学与病因；

✔️ 解释这些疾病的临床表现、发病机制与治疗策略；

✔️ 讨论疾病预防、风险降低及治疗策略；

✔️ 运用批判性思维设计公共卫生政策与疾病管理策略。

# BMS8106
## 干细胞与再生医学

## **📘 简介**
干细胞是未分化的生物细胞，具有分化为不同类型细胞的潜力，因此具有替代体内退化细胞的能力，有望重建由于疾病或衰老导致的器官系统功能。该课程将深入介绍干细胞生物学和再生医学领域的最新进展，涵盖干细胞和基于干细胞的组织再生的基本科学知识和技术概念。学生将探讨人体正常修复和再生过程的基本原理，分析影响组织修复和再生的调节过程，并将细胞与分子生物学、组织工程和遗传学等已有知识结合应用，分析这些过程。

## **🎯 学习目标**

完成课程后，学生将能够：


✔️ 区分人体不同组织修复和器官再生的模式；

✔️ 发现干细胞修复、再生和替代生物学中的关键分子和调节因素；

✔️ 比较分子疗法、干细胞疗法、生物启发材料和新型生物材料的操作原理；

✔️ 理解并评估当前文献，分析这些新兴技术的生物学功能性、生物相容性及微纳技术的应用；

✔️ 评估各种方法在操控人体再生过程中的应用。

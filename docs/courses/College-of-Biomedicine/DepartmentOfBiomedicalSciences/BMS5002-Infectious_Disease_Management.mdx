# BMS5002
## 传染病管理

## **📘 简介**
本课程提供学生关于传染病诊断、控制和预防的原则与实践。课程涵盖疾病原理、诊断、分子流行病学、疾病传播机制及流行病学原理，重点讨论传染病在香港、亚洲及全球的影响与防控。还将涉及抗生素耐药性、疫苗学、公共卫生政策和应急响应等内容。

## **🎯 学习目标**

完成课程后，学生将能够：


✔️ 理解传染病的基本生物学原理和流行病学；

✔️ 描述主要传染病的传播模式与致病机制；

✔️ 讨论不同传染病的控制、评估策略和应急响应；

✔️ 运用批判性思维分析公共卫生政策及疾病管理。

|----------------------|---------|-----------------------------------------------------------------------------------------------|

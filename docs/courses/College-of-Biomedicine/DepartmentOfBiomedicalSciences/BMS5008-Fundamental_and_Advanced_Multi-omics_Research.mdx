# BMS5008
## 基础与高级多组学研究

## **📘 简介**
本课程通过讲座和实践帮助学生理解各种组学方法的基本原理和应用。课程内容包括（1）常用测序平台的原理；（2）基因组测序、转录组学及其应用；（3）表观遗传学修改与调控；（4）转录后调控及其在生物学中的作用；（5）蛋白质组学的最新进展；（6）多组学数据分析及综合功能基因组学；（7）基因编辑技术与基因组操控；（8）医学和治疗实施中的新兴挑战。

## **🎯 学习目标**

完成课程后，学生将能够：


✔️ 描述常见测序技术的基本概念和应用；

✔️ 解释表观遗传学、转录组学和蛋白质组学的最新进展；

✔️ 运用批判性思维分析最新的多组学研究，设计或改进实验；

✔️ 讨论基因编辑、基因组操控及其在医学中的潜力与挑战。

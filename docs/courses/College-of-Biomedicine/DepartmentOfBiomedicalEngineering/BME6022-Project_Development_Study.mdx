# BME6022
## 项目开发研究

## **📘 简介**
本课程旨在培养学生在先进技术系统、机械工程或生物医学工程相关领域开展项目开发研究的能力。学生将学习制定项目工作计划的技能，包括战略规划、团队合作和项目管理，并为满足特定需求制定研发方案。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/BME6022.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/BME6022.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 理解研发工作的范围、性质及调查过程；

✔️ 根据选定的工程主题制定研发提案。

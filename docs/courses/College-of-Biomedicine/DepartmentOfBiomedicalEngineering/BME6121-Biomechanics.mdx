# BME6121
## 生物力学

## **📘 简介**
本课程为学生介绍开发生物医学假体设备所需的生物力学基本概念及其在解决相关工程问题中的应用。通过合作学习和实验，学生将掌握相关的工程技术知识。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/BME6121.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/BME6121.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述生物力学的基本概念及其对物体受力行为的影响；

✔️ 阐释生物材料和生物医学设备中的工程问题并解决它们；

✔️ 应用生物力学知识分析生物医学产品的结构和功能行为；

✔️ 展示团队项目的研究背景、方法及结论。

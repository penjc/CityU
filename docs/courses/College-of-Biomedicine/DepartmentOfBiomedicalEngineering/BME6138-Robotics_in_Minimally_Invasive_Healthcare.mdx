# BME6138
## 微创医疗机器人

## **📘 简介**
随着现代医疗对微创或非侵入式诊断和治疗的需求增加，医疗机器人技术为患者提供了减轻痛苦和加速康复的可能。本课程涵盖医疗机器人系统在微创医疗中的历史、最新发展及应用。内容包括达芬奇手术系统、机器人导管与内镜、可吞服胶囊、实验室芯片/器官芯片设备、AI在医疗中的应用，以及微/纳米机器人。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述微创医疗机器人系统的基本概念和目标；

✔️ 解释代表性机器人系统的设计、工作原理及应用；

✔️ 阐释 AI 和机器学习在医疗机器人中的应用；

✔️ 分析现有技术的挑战并评估潜在解决方案；

✔️ 提出系统级整合策略，设计创新的医疗机器人系统。

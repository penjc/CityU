# BME6115
## 生物机器人

## **📘 简介**
本课程探讨生物启发的机器人系统和生物医学机器人，从自然界汲取灵感并将其应用于工程化系统。学生将学习动力学、比例物理、传感器、生物力学和运动等基本原理，并通过实例了解生物解决方案如何转化为机器人系统。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 比较生物启发机器人和生物医学机器人的特点；

✔️ 掌握飞行、行走等运动的动力学原理；

✔️ 分析和设计生物机器人系统；

✔️ 将生物原理创新性地应用于工程系统。

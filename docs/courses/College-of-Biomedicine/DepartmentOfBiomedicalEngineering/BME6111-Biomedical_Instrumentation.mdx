# BME6111
## 生物医学仪器

## **📘 简介**
本课程介绍生物医学仪器的基本概念、工作原理和设计方法，涵盖当前工程和医学中的相关应用。课程内容包括非侵入式传感器、可穿戴设备、血压测量设备、心脏起搏器、除颤器以及人工耳蜗等，同时探讨现有系统的挑战以及从自然界获得灵感的设计方案。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述生物医学仪器系统的基本概念；

✔️ 讨论重要部件的工作原理；

✔️ 设计创新的生物医学传感器和仪器；

✔️ 应用系统级集成原则设计多功能仪器系统。

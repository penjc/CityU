# BME6114
## 高级控制系统

## **📘 简介**
本课程深入讲授线性时不变系统建模、状态空间分析与设计，以及稳定性分析和反馈控制设计工具，并简要介绍非线性系统控制、自适应控制与最优控制等高级主题。课程内容以数学为基础，结合一般工程系统的实例。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/BME6114.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/BME6114.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 理解系统与控制的基本概念；

✔️ 使用状态空间方法分析系统；

✔️ 为工程系统设计反馈控制律；

✔️ 将高级控制理论应用于实际问题。

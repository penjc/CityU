# BME6123
## 灵活生物电子学及其医疗应用

## **📘 简介**
灵活生物电子学是将多学科原理应用于生物学、医学和人类健康护理的技术。由于生物组织的柔软性，现有诊断和治疗方法在舒适性、侵入性和便捷性方面存在挑战。灵活生物电子学技术通过结合“软”材料和优化设备设计，推动创新设备如皮肤传感器的发展，以便于疾病的预防、诊断、监测和治疗。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述灵活生物集成电子学的基本背景；

✔️ 解释与生物医学相关的感测参数基本概念；

✔️ 理解生物电子学中的材料学、化学、物理和力学原理，以及如何将这些原理应用于新型软生物传感技术的开发；

✔️ 选择现有的生物医学感测、诊断或治疗问题，并提出解决方案。

# BME5111
## 再生医学

## **📘 简介**
本课程作为临床医学、生物医学工程和发育生物学的交叉学科，涵盖了用以替换、工程化或再生人类细胞、组织或器官的技术，旨在恢复或建立正常功能。学生将学习从干细胞和组织工程到临床应用（如骨骼和软骨再生）的核心知识。

🔗 **相关链接**

- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/BME5111.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述干细胞生物学和组织工程学的基本原理；

✔️ 解释干细胞生物学和组织发育的基本概念；

✔️ 评估设计和开发组织培养系统的现有技术；

✔️ 确定实施中的实际问题；

✔️ 针对特定组织工程问题设计可行的工程方案。

# BME5110
## 生物医学工程设计

## **📘 简介**
本课程为无生命科学背景的学生提供生物医学工程的广泛概述，重点是医疗保健的共同主题，包括基础人体生理学、病理学、生物医学技术与成像、分子成像，以及医疗设备和技术管理。学生将学习生物医学工程的基本概念和工具，并应用这些知识解决医疗行业相关问题。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 阐释生物医学技术的基本概念及其与人体生理的关系；

✔️ 分析人体系统的基本工作原理及其与病理和医疗设备的关系；

✔️ 设计医疗保健和疾病管理方案；

✔️ 应用生物医学工程知识促进健康与医疗质量。

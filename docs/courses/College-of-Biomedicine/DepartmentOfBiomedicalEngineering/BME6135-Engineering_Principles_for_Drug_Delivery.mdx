# BME6135
## 药物输送的工程原理

## **📘 简介**
本课程通过工程原理和材料科学探讨药物输送的基础和应用。课程内容包括药物输送的基本原理、传输机制、材料与制剂，以及其在生物医学中的应用。学生将通过讲座和项目学习，掌握减少毒性、提高药效和改进药物使用的方法。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 阐释药物输送的基本概念与临床需求；

✔️ 分析人体内不同药物输送系统的工作原理；

✔️ 选择适合临床有效输送的材料与设备；

✔️ 设计实现有效药物输送的制剂和设备。

# BME5108
## 人机界面

## **📘 简介**
本课程旨在教授人机界面的历史和最新技术，以及人与机器之间的交互方法。学生将学习如何结合自动化设备与数学和组织工具，以创建复杂系统。课程还涵盖用户界面设计的科学方法和评估标准，帮助学生缩短设计周期并降低成本。

🔗 **相关链接**

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 阐释人机界面的基本概念及设计机制；

✔️ 理解科学方法及设计原则，并应用于不同控制对象；

✔️ 设计符合用户需求的界面，并通过超媒体实现自动控制；

✔️ 应用标准方法评估人机界面的性能。

---

## **📊 评估方式**

| **评估项目**                 | **权重** | **具体描述**                                                                                       |
|-----------------------------|---------|---------------------------------------------------------------------------------------------------|
| **📝 小组展示及同伴评估**      | 25%     | 通过小组展示和同伴评估展示对课程目标的理解及团队协作能力。                                           |
| **📄 个人报告**               | 25%     | 学生需完成个人报告，展示其独立研究能力和对课程目标的掌握。                                           |
| **📄 考试**                   | 50%     | 2小时书面考试，测试学生对人机界面设计和评估的综合能力。                                              |

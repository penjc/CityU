# BME6141
## 单分子生物物理学的基础与快速诊断应用

## **📘 简介**
本课程深入探讨单分子水平的生物物理学原理及其在快速诊断中的应用。内容包括单分子动力学与光谱学、蛋白折叠、光学/磁光镊技术，并扩展至生物医学中的实际应用，如快速诊断、超灵敏检测、高通量测序及即时诊断等。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述单分子生物物理学的基本概念与技术；

✔️ 关联生物物理学原理至快速诊断和医学应用；

✔️ 评估最新研究进展并应用于生物医学工程研究中。

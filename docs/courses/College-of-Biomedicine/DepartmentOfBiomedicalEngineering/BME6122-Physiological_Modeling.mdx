# BME6122
## 生理建模

## **📘 简介**
本课程将介绍生物电、生物力学和生物声学活动的数学模型，涵盖神经肌肉系统、心血管系统、听觉系统等生理系统。课程的目的是帮助学生获得对人体生理过程的理论理解。主要内容包括神经肌肉系统的结构与功能、生物神经网络基础、肌电通道性能分析等。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述与生理系统相关的基本概念；

✔️ 讨论生物电信号传播机制及其在神经肌肉系统中的应用；

✔️ 建模并解释神经肌肉、心血管和听觉系统的各种组件；

✔️ 将生理系统的建模应用于生物医学仪器的设计和仿生信号处理技术。

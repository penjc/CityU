# BME6137
## 医学诊断

## **📘 简介**
本课程为学生提供疾病预测、诊断、预防及最佳治疗方案规划的全面概述。重点包括分子影像技术（如 PET、SPECT 和分子 MRI）、目标探针、肿瘤及传染病中的分子影像，以及分子诊断技术（如核酸技术、OMICS 及精准医疗）。课程还涵盖放射组学和人工智能在诊断中的基础应用，帮助学生为未来分子诊断领域的进一步探索奠定坚实基础。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 理解医学诊断的基本原理，并应用于癌症和传染病诊断；

✔️ 阐释常见分子诊断测试和影像技术的基础知识；

✔️ 讨论分子诊断在临床中的实际应用；

✔️ 评估当前文献并撰写相关科学报告。

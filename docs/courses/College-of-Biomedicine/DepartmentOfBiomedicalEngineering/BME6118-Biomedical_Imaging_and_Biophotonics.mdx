# BME6118
## 生物医学成像与生物光子学

## **📘 简介**
本课程旨在介绍医学成像技术和生物光子学的基本原理及其在实际设备中的应用。课程内容包括磁共振成像、X射线计算机断层扫描、超声波成像、光散射理论、光学传感与光谱、光学显微镜以及光声断层成像。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述主要医学成像技术的概念和原理；

✔️ 运用光子与生物组织的相互作用概念分析问题；

✔️ 解释光学传感、光谱和成像技术的基本原理；

✔️ 根据不同应用选择适当的成像技术；

✔️ 讨论医学和生物光声成像技术的应用。

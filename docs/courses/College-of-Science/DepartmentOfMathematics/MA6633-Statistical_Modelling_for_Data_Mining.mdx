# MA6633
## 数据挖掘的统计建模 (Statistical Modelling for Data Mining)

## **📘 简介**
本课程旨在为学生提供数据挖掘和机器学习技术的数学基础与统计知识。重点介绍学习、分类及预测的统计属性与分析，包括贝叶斯决策理论、模型选择、线性与非线性分类模型、决策树及聚类。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 阐述数据挖掘中的基本问题与数学挑战；

✔️ 使用监督学习与非监督学习算法，理解其基本原理；

✔️ 选择适合的数据挖掘模型解决实际问题。

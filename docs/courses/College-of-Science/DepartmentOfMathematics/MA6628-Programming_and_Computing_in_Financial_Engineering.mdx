# MA6628
## 金融工程中的编程与计算 (Programming and Computing in Financial Engineering)

## **📘 简介**
本课程旨在发展学生应用数值方法分析和构建金融产品模型的能力，讲解如何通过优化算法和计算机编程实现金融和保险模型。学生将学习求解偏微分方程的基本方法，以及如 Monte Carlo 模拟和有限差分法等数值技术。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/MA6628.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/MA6628.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述金融计算中的基本问题与框架；

✔️ 清晰解释解决偏微分方程的基本方法；

✔️ 实施数值与计算方法（如有限差分法、Monte Carlo 模拟）以评估复杂金融问题；

✔️ 设计并实现编程解决方案，评估金融风险并构建金融模型；

✔️ 应用数值方法分析金融现象及实际问题。

# MA6632
## 金融大数据的统计分析 (Statistical Analysis of Financial Big Data)

## **📘 简介**
本课程教授金融大数据的统计分析技术，涵盖假设检验、回归模型、多元分析及时间序列预测。学生将学习使用统计软件完成数据分析，为科学与工程问题提供解决方案。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/MA6632.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/MA6632.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 执行数据集的假设检验并推断潜在分布；

✔️ 构建统计模型和实验设计；

✔️ 应用多元分析方法解决复杂数据问题；

✔️ 使用统计方法进行产品质量评估与时间序列预测；

✔️ 熟练使用统计软件完成数据分析。

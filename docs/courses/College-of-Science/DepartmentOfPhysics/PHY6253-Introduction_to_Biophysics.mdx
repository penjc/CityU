# PHY6253
## 生物物理学导论 (Introduction to Biophysics)

## **📘 简介**
本课程介绍了生物物理学这一交叉学科。内容包括分子和细胞生物学的基础知识，以及扩散理论（在细胞行为中很重要）、热力学（用于理解蛋白质等大分子的功能）和三种用于研究生物的物理方法：X 射线晶体学、冷冻电子显微镜 (Cryo-EM) 和分子动力学模拟。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 理解 DNA、RNA 和蛋白质在细胞中的化学组成及其功能；

✔️ 掌握扩散理论及其在细胞行为中的作用；

✔️ 理解热力学及其在生物大分子结构与功能中的作用；

✔️ 掌握 X 射线晶体学、Cryo-EM 和分子动力学模拟等生物物理学方法；

✔️ 进行文献研究并进行学术展示，或完成计算机模拟并提交报告。

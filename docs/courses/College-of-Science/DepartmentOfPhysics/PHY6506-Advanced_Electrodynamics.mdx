# PHY6506
## 高级电动力学 (Advanced Electrodynamics)

## **📘 简介**
本课程旨在为研究生提供进行电动力学相关领域研究所需的高级概念与数学方法。课程涵盖电磁波在均匀材料中的传播、表面波行为、等离激元、波导、腔体、散射和辐射现象，还包括光子晶体和超材料等相关应用。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别电动力学的重要性及其应用领域；

✔️ 理解电动力学的核心概念和物理机制；

✔️ 应用分析方法解决实际问题；

✔️ 开发具有特定波动特性的电动力学系统。

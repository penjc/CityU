# PHY6502
## 高级计算方法 (Advanced Computational Methods)

## **📘 简介**
本课程涵盖跨学科科学中涉及的模拟与建模方法，目标是通过理论讲解和软件应用指导学生完成发现导向的项目，例如新材料设计、新物理探索、DNA 修复、药物设计或金融市场预测。

- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/PHY6502.pdf)

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 认识模拟和建模在跨学科科学中的重要性；

✔️ 使用工具解决跨学科问题并进行模拟与建模；

✔️ 阐明影响模拟与建模质量的关键变量；

✔️ 应用方法解决材料科学、物理、化学、生命科学和金融领域的问题；

✔️ 完成发现导向的项目，例如新材料设计或药物设计。

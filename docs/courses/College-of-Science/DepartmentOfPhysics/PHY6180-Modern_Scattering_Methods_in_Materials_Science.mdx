# PHY6180
## 材料科学中的现代散射方法 (Modern Scattering Methods in Materials Science)

## **📘 简介**
本课程涵盖材料科学中涉及 X 射线和中子散射的实验和应用物理学主题。重点包括在实验室及大型设施（如同步加速器和中子源）中进行散射实验的基本原理、技术和应用，帮助学生探索先进材料的研究和创新。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 解释现代散射技术及其在材料研究中的应用；

✔️ 掌握不同散射技术的基本知识，特别是中子衍射；

✔️ 比较 X 射线与中子散射的异同；

✔️ 理解散射的基本理论及其在材料结构研究中的应用；

✔️ 运用动力学衍射理论解决材料科学问题；

✔️ 掌握小角 X 射线散射 (SAXS) 和小角中子散射 (SANS) 的基础知识；

✔️ 描述非弹性中子散射 (INS) 和准弹性中子散射 (QENS) 的基本原理及应用。

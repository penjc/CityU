# PHY6252
## 统计力学 (Statistical Mechanics)

## **📘 简介**
本课程旨在让研究生掌握统计力学的知识，这些知识对凝聚态物理研究和相关文献的理解至关重要。课程内容从统计力学的基本概念开始，涵盖弱相互作用系统、强相互作用系统，并引入波动-耗散定理及耗散系统的其他相关知识。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 正确使用重要的技术术语和定义；

✔️ 使用适当的数学符号，将统计力学定律应用于现代物理问题的研究；

✔️ 运用统计力学定律解决现代物理学问题；

✔️ 通过识别潜在的物理规律，分析并解决实际或假设的统计物理问题。

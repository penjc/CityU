# PHY6521
## 高级固体物理 (Advanced Solid State Physics)

## **📘 简介**
本课程旨在为学生提供固体物理的高级知识，帮助学生理解当代文献并从事凝聚态物理和材料科学的前沿研究。内容涵盖晶体的结构与电子特性、金属的热力学和输运性质、磁性、超导性、拓扑能带理论及贝里相位效应等主题。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 掌握晶体学和固体中电子结构的基本知识，理解金属与绝缘体的根本区别；

✔️ 推导 Bloch 定理，并将其应用于分析能带结构及金属的热力学和输运性质；

✔️ 计算贝里相位并分析绝缘体的拓扑性质；

✔️ 了解量子霍尔效应、边缘态、拓扑绝缘体及贝里相位效应在金属中的作用。

# SM5313
## 互动媒体 II (Interactive Media II)

## **📘 简介**
本课程将多媒体创作与计算机编程相结合，帮助学生在特定主题领域（如数字文学或行为艺术）中创建更高级的互动项目。学生将学习使用开源编程环境 Processing 和硬件设备 Arduino，通过创新的界面策略设计多层次用户体验。

🔗 **相关链接**

- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/SM5313.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别互动艺术项目中的基本概念与流程；

✔️ 讨论项目从概念到实现的发展过程；

✔️ 解释与数字媒体项目相关的人机交互机制；

✔️ 设计项目展示与提交的方法；

✔️ 从仿真到装置设计一个原型互动艺术项目。

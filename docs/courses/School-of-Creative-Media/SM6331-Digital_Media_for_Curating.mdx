# SM6331
## 策展中的数字媒体 (Digital Media for Curating)

## **📘 简介**
本课程提供关于策展中使用的数字媒体及其技术的概览，帮助学生理解数字媒体如何支持展览的教育与交流目标。课程涵盖在线和离线媒体的使用及其优劣势分析，教授学生如何选择、计划和生产适合不同展览需求的数字媒体，并结合经典媒体和数字技术实现策展目标。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述数字媒体的应用场景和发展历史；

✔️ 分析数字与传统媒体在策展中的优缺点；

✔️ 理解不同访客群体的需求并设计参与性展览；

✔️ 探索策展过程中数字媒体的策略性应用；

✔️ 计划、计算并生产数字媒体以实现策展目标。

## **📊 评估方式**

| **评估项目**                 | **权重** | **具体描述**                                                                                       |
|-----------------------------|---------|---------------------------------------------------------------------------------------------------|
| **📂 阅读总结**             | 10%      | 总结指定阅读内容并提供见解。                                                                        |
| **📋 博物馆访问记录**         | 15%      | 提交访问记录并记录策展人关于数字媒体应用的讨论结果。                                                   |
| **📂 数字媒体制作计划**       | 40%      | 制定数字媒体策展计划，包括预算和时间表。                                                             |
| **📋 结果展示与讨论**         | 15%      | 对成果进行展示与讨论，接受反馈并调整策略。                                                           |
| **📂 手册文章**              | 20%      | 撰写关于数字媒体在策展中的文章，展示个人研究成果。                                                     |

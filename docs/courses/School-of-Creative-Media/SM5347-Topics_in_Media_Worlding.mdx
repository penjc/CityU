# SM5347
## 媒体世界化专题 (Topics in Media Worlding)

## **📘 简介**
本课程提供对“世界化”作为跨越视听、传统与当代媒体的一种包容性横向设计媒体平台的全新理解。学生将学习数字摄影、电影、视频、计算机游戏、音频和声音艺术的演变，从传统美学到未来媒体形式的转变。通过实验和练习，学生将获得关于虚拟现实、增强现实、扩展电影和声音体验的技术理解，并探索新的创作可能性。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 理解并培养媒体世界化的新趋势；

✔️ 独立创建艺术与设计实践的媒体概念；

✔️ 探索数字电影和声音作为时间媒介的可能性；

✔️ 使用最新数字技术拓展媒体世界化的范畴；

✔️ 通过影像、声音和其他媒体技术演变传统方法；

✔️ 将艺术创造力与技术知识结合，创作具有个人风格的作品。

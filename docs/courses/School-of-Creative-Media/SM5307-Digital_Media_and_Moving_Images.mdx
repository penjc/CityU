# SM5307
## 数字媒体与动态影像 (Digital Media and Moving Images)

## **📘 简介**
本课程旨在通过实验性创作探索数字媒体与动态影像的艺术形式。学生将学习使用 Processing 和 Max/MSP/Jitter 等程序语言作为艺术媒介，设计和开发自己的创意项目，并研究实验影像、新媒体艺术、计算机音乐和媒体表演等多学科内容。课程重点在于结合技术与艺术形式，推动跨领域创作。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述计算机编程在动态影像和视听媒体中的基础概念；

✔️ 在艺术创作中应用数字媒体和计算技术；

✔️ 识别数字视听艺术的特点；

✔️ 使用算法技术创作独特风格的艺术作品。

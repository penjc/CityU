# SM6302
## 毕业项目 - 工作坊 II (Thesis Project - Studio II)

## **📘 简介**
本课程是MFA项目中毕业论文工作坊的最终部分。学生将在导师的个别指导下完成创意项目的研究、原型设计、制作与展示。课程形式包括导师指导、小组教学和同行评审，旨在提供一个强化的平台，促进创意交流、媒介探索和个人艺术发展的深化。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 与导师讨论毕业项目意图并完成创意项目；

✔️ 撰写反映个人创作实践的艺术家声明，结合更广泛的社会文化背景；

✔️ 设计记录创作过程的创意作品集；

✔️ 在同行评审中讨论并完善最终项目；

✔️ 基于最终作品展示创新美学、创意理念和制作技术。

# SM5335
## 新媒体艺术的考古学 (Archaeology of New Media Art)

## **📘 简介**
本课程基于米歇尔·福柯的“系谱学”和“考古学”理论，探讨新媒体艺术的批判修订史。通过媒体考古学方法，分析录制和展示技术的演变对艺术创作的意义。学生将学习如何批判性地反思新媒体艺术的基本问题，并探讨艺术与技术之间的互动。课程涵盖实验电影、互动艺术、信息技术艺术和艺术与科学的交叉等主题。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 阐释媒体考古学和早期技术发展的社会影响；

✔️ 反思媒体对艺术作品的影响，探索新媒体带来的形式和意义创新；

✔️ 探讨媒体艺术的复杂性和多样性；

✔️ 从媒体考古学的角度进行独立研究。

# SM6317
## 媒体研究中的研究项目 (Research Project in Media Studies)

## **📘 简介**
本课程是MACM项目的最终学习成果，要求学生提出并完成一个小规模的历史或批判性研究项目，研究主题需与课程内容相关。项目可为新发起，也可经导师批准后延续之前的研究工作。学生将在导师的指导下，通过一系列活动完成研究，并提交最终成果。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述并运用关键概念和术语分析相关研究主题；

✔️ 使用多种研究方法整合人文学科研究成果；

✔️ 以对媒体研究领域有显著贡献的方式呈现研究结果。

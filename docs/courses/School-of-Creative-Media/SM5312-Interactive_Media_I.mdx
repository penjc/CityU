# SM5312
## 互动媒体 I (Interactive Media I)

## **📘 简介**
本课程涵盖互动多媒体创作的概念与技术基础，旨在帮助学生评估、理解并创造引人入胜的互动多媒体作品。课程将教授软件、硬件和用户体验等多种媒体的制作与呈现方法，同时探讨技术、创造力和社会的关系。

🔗 **相关链接**

- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/SM5312.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别互动艺术实践的类别；

✔️ 掌握互动艺术的历史与理论知识；

✔️ 发现互动媒体艺术作品的开发与创作方法；

✔️ 创建具有个人风格的互动多媒体项目原型。

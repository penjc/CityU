# SM5348
## 创意媒体策展 (Curating Creative Media)

## **📘 简介**
本课程探索策展与数字技术的交叉，重点分析数字技术对艺术创作、传播和接受的影响。课程涵盖扩展策展的历史、理论与实践，研究区块链、AI、机器学习和 Web 3 等新兴技术对策展实践、观众参与和数字艺术传播的影响。学生将了解数字技术如何改变策展人与艺术家及观众之间的互动模式。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 探索数字技术对策展实践、观众参与和艺术欣赏的影响；

✔️ 开展技术文化研究，识别相关领域的最新趋势；

✔️ 学习选择、组织和管理创意资产的方法；

✔️ 掌握展示技术以推广文化内容和创意资产；

✔️ 理解扩展策展的历史背景；

✔️ 使用理论与实践知识策展创意媒体项目。

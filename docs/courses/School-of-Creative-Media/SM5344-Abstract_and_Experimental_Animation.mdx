# SM5344
## 抽象与实验动画 (Abstract and Experimental Animation)

## **📘 简介**
通过一系列放映、工作坊和讲座，本课程介绍抽象与实验动画的历史与当代实践。课程强调声音和音乐在动态影像中的重要性，探索视觉音乐的概念，以及节奏、时间与对位如何构建或打破抽象运动的阅读。学生将通过短期项目和练习，学习设计和创作抽象与实验动画作品，作为艺术表达形式和时间媒介的概念工具。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 理解抽象与实验动画的历史与实践；

✔️ 通过小组作业掌握动画技能与技术；

✔️ 运用抽象动态影像表达声音、音乐及概念；

✔️ 在数字或模拟动画中进行实验并接受创意挑战；

✔️ 将基本技术能力转化为独特的个人风格。

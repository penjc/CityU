# SM5337
## 超越人类世的美学 (Aesthetics Beyond the Anthropocene)

## **📘 简介**
本课程旨在为学生提供关于艺术和人类世的当代理论的基础知识，重点学习关键文本及其历史背景与话语定位。学生将通过阅读、写作和批判性分析提升论证能力，同时探讨理论如何融入策展实践。通过研究近期展览，学生将获得在项目中运用理论的技能。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述生态人文学科的主要问题和概念；

✔️ 掌握生态艺术实践领域的关键理论和方法；

✔️ 对理论问题进行批判性分析并撰写分析性文章；

✔️ 将理论概念整合到策展项目中。

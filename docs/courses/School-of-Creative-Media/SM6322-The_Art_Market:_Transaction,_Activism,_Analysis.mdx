# SM6322
## 艺术市场：交易、行动与分析 (The Art Market: Transaction, Activism, Analysis)

## **📘 简介**
本课程探讨国际和本地艺术市场的运作机制和策略，并从哲学的角度分析价值与意识形态的关系。学生将研究艺术市场的历史、参与者和经济体系，以及艺术家如何操控与批判金钱和价值的主题。课程涵盖传统艺术、新媒体艺术、参与性艺术和计算艺术等不同领域。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 理解当代国际艺术市场的起源、社区与角色；

✔️ 分析并发展本地艺术市场的理论见解；

✔️ 探讨支持或批判金钱、价值与艺术市场的艺术家；

✔️ 描述并理论化将艺术商品化的方法与话语；

✔️ 通过小组讨论和自主学习，不断完善研究见解并生成研究材料。

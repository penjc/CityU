# SM5343
## 法律、政策与全球媒体平台 (Law, Policies and Global Media Platforms)

## **📘 简介**
本课程探讨与内容监管相关的最具争议性问题，包括虚假信息、算法化审查、数据资本主义、数字监控、数字版权和AI生成艺术等主题。通过研究全球平台的法律和政策框架（如美国、中国、欧盟和俄罗斯的主要平台），学生将理解数字时代对文化、权利、民主与和平的影响，并学习保护隐私、版权和言论自由的实用技巧。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述和解释数字文化、法规和社会秩序相关的关键现象和趋势；

✔️ 运用研究论文中的关键概念分析数字文化与社会秩序的交互关系；

✔️ 批判性评估数字文化和法规交互所引发的问题与趋势，并提供解决方案。

# SM6332
## 计算机游戏与社会 (Computer Games and Society)

## **📘 简介**
本课程探讨计算机游戏如何成为社会活动的平台，超越了传统的游戏空间。学生将分析游戏中的玩家与开发者在模组制作、内容编写、组织活动和社交等方面的角色，研究游戏设计如何响应玩家社区的需求。课程强调通过阅读相关研究文献和分析元游戏现象，帮助学生理解计算机游戏在社会和文化中的深远影响。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 应用分析社会和计算机游戏相关活动的适当方法与技巧；

✔️ 描述计算机游戏及其相关现象对社会的影响；

✔️ 使用相关理论和术语反思计算机游戏与社会问题。

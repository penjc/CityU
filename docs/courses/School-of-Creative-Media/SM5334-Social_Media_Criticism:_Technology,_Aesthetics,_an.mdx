# SM5334
## 社交媒体批评：技术、美学与文化 (Social Media Criticism: Technology, Aesthetics, and Culture)

## **📘 简介**
本课程探讨以社交媒体为主导的视听平台（如 YouTube、TikTok）的技术、美学与文化影响。学生将超越对社交媒体的悲观或乐观观点，研究其在数据流媒体基础设施、界面设计、算法推荐和文化分析方面的演变。此外，学生将通过学期项目创建视听作品，以批判性地反映社交媒体作品及其相关的亚文化。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述视听主导社交媒体的媒介特性；

✔️ 分析社交媒体对技术、美学与文化的影响；

✔️ 将社交媒体批评融入视听媒介，并结合理论与技术；

✔️ 创作基于研究发现的视听艺术作品。

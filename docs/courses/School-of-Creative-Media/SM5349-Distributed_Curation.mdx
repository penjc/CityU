# SM5349
## 分布式策展 (Distributed Curation)

## **📘 简介**
本课程探讨突破传统博物馆框架的创新艺术展示方式，旨在促进多样性、公平性和包容性。通过研究经典案例，学生将了解分布式策展模型如何在虚拟与物理空间中运作，以及投票和众包等民主化策展方式的潜力。本课程还将探讨艺术家作为策展人的角色及其机遇与挑战，最终帮助学生策划并实现分布式项目。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 利用非传统技术赋权边缘化群体，促进包容与参与；

✔️ 理解新兴展示形式和社会实践的理论与实际背景；

✔️ 探索分布式策展模型在多样性、公平性和包容性中的角色；

✔️ 分析虚拟与物理空间中使用的各种分布式策展模型；

✔️ 理解投票、众包等互动模型在策展中的民主化潜力；

✔️ 探讨艺术家作为策展人的角色及其面临的挑战和机遇；

✔️ 利用理论和实践知识策划分布式项目。

# SM6300
## 毕业项目 - 工作坊 I (Thesis Project - Studio I)

## **📘 简介**
本课程是MFA项目四部分工作坊与论文要求中的第三部分。学生将通过个别指导、小组教学和同行评审，完成创意项目的研究、原型设计、制作与展示。课程目标是为学生提供一个强化的平台，以实现创意交流、媒介探索与个人艺术发展的深化。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/SM6300.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/SM6300.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 在工作坊项目中运用高级媒介美学与制作技术；

✔️ 基于研究和实验展示创意项目；

✔️ 设计并解释项目进展，用于同行评审和反馈；

✔️ 在CILO 2的基础上讨论创新美学、创意理念及制作技术。

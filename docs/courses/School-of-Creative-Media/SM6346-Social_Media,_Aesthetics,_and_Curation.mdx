# SM6346
## 社交媒体、美学与策展 (Social Media, Aesthetics, and Curation)

## **📘 简介**
本课程探讨社交媒体与当代艺术策展的交叉领域，分析数字平台如何改变艺术创作、传播与接受方式。学生将研究社交媒体对策展实践的影响，并理解其在当代艺术中的角色。课程强调社交媒体的媒介特性，鼓励学生将Instagram、VR和视频等平台视为策展空间，并结合理论知识与实际技能，完成社交媒体策展项目。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 理解社交媒体策展的媒介特性；

✔️ 分析社交媒体在当代艺术领域的发展历程；

✔️ 阐述社交媒体对美学的影响；

✔️ 学习与社交媒体生态系统相关的策展技术与技能；

✔️ 运用理论与实践知识完成策展项目。

# SM6347
## 去中心化策展的协议与技术 (Protocols and Techniques for Decentralised Curation)

## **📘 简介**
本课程探讨策展与去中心化技术的交叉领域，研究去中心化技术（如区块链、NFTs、算法模型）对艺术创作、传播和接收的影响。学生将分析去中心化对策展实践、观众参与及艺术流通的影响，并学习如何设计策展协议，结合理论与实践知识，通过算法策展和交易艺术实现创新。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 设计多样化、包容性的策展平台，运用去中心化策展方法；

✔️ 理解艺术的代币化，并运用虚拟艺术形式吸引观众；

✔️ 描述交易艺术对当代美学观念的影响；

✔️ 掌握去中心化技术与策展实践相关的技巧与技术；

✔️ 完成结合理论与实践知识的策展项目。

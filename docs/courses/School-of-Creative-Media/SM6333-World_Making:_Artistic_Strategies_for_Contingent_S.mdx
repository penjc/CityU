# SM6333
## 世界创生：面向偶发系统的艺术策略 (World Making: Artistic Strategies for Contingent Systems)

## **📘 简介**
本课程探讨世界创生作为一种艺术实践，研究其在网络文化与系统艺术中的应用。学生将学习如何从哲学和文化批判的角度理解当代艺术中的偶发系统，以及这些系统如何重新定义感知世界的方式。课程结合理论与实践，涵盖从实验艺术到建筑、表演和虚拟场景等多领域的创作，帮助学生建立对世界创生的全面理解。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 批判性地识别世界创生作为一种艺术实践；

✔️ 理解新媒体研究中世界创生的核心概念；

✔️ 应用课堂所学与创意合作者共同完成世界创生项目；

✔️ 在不同情境下创意性地表达相关概念与信息。

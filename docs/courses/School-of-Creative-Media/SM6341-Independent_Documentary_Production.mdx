# SM6341
## 独立纪录片制作 (Independent Documentary Production)

## **📘 简介**
本课程提供独立纪录片制作的实践机会。学生将学习制作10-15分钟纪录片的各个步骤，包括前期调研、制作规划、撰写大纲、拍摄、剪辑以及试映。课程注重培养学生与不同社会文化背景的拍摄对象建立积极关系的能力，并探索在有限使用摄影机的情况下如何有效进行记录。还将介绍材料组织、采访和叙事技巧，为未来非虚构作品的制作打下基础。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 确定拍摄主题、对象和制作计划；

✔️ 按照纪录片拍摄计划完成制作各阶段，并克服拍摄过程中的各种挑战；

✔️ 发现拍摄过程中的重要问题，包括行为、伦理和突发事件；

✔️ 将分散的素材组织成完整的叙事或创新的纪录片作品。

## **📊 评估方式**

| **评估项目**             | **权重** | **具体描述**                                                                                       |
|-------------------------|---------|---------------------------------------------------------------------------------------------------|
| **📂 制作计划与反思**     | 25%     | 制定完整的制作计划并撰写反思报告。                                                                   |
| **📋 拍摄报告与素材展示** | 15%     | 展示拍摄的素材并提交拍摄报告。                                                                     |
| **📂 试映讨论与反馈**     | 10%     | 参与试映讨论并提供反馈意见。                                                                       |
| **📂 撰写大纲与剪辑**     | 25%     | 完成纪录片的大纲撰写与初步剪辑。                                                                     |
| **📋 最终版本展示**       | 25%     | 提交最终版本纪录片并参与课堂讨论。                                                                   |

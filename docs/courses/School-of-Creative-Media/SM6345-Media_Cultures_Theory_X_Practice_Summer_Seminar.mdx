# SM6345
## 媒体文化理论与实践夏季研讨 (Media Cultures Theory × Practice Summer Seminar)

## **📘 简介**
本课程旨在将文化研究、美学及生产研究的批判理论与新媒体知识转化为实际的专业实践。课程形式为研究生研讨会，通过具体的文化生产案例（如创意产业、艺术机构、文化组织）分析理论与实践的多向关系。学生将研究这些实践如何受到理论影响，反过来又如何丰富理论视角。本课程强调在文化生产的实际背景下操作理论知识，培养学生在社会、媒体、法规和城市环境中进行干预的能力。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 确定文化研究、美学及批判理论在文化生产中的关键概念与角色；

✔️ 阐明这些概念如何在实际背景下操作；

✔️ 在独立研究、批判写作及艺术实践中应用这些概念；

✔️ 明确理论知识在未来职业活动中的价值。

## **📊 评估方式**

| **评估项目**                       | **权重** | **具体描述**                                                                                       |
|-----------------------------------|---------|---------------------------------------------------------------------------------------------------|
| **📂 阅读展示与论文**              | 25%     | 针对指定阅读材料进行展示，并撰写分析论文。                                                             |
| **📋 反应性写作与课堂讨论**          | 15%     | 结合课堂内容与阅读材料进行反思性写作和讨论。                                                           |
| **📂 个人观察日志**                | 25%     | 对文化机构/艺术活动进行观察，并通过两阶段日志展示理论与实践的联系。                                            |
| **📋 期末论文/项目**               | 35%     | 提交结合课程概念的最终研究论文或创意项目。                                                               |

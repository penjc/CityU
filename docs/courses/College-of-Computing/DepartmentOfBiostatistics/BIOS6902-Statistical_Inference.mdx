# BIOS6902
## 统计推断

## **📘 简介**
本课程基于概率论的基本原理（建议学习 BIOS5800 作为基础），全面讲授点估计、区间估计和假设检验，包括功效计算和似然比检验等内容。通过理论与实践的结合，为生物医学及公共健康领域中的数据建模和推断奠定基础。

- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/BIOS6902.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 理解统计推断的基本技术；

✔️ 构建并拟合应用于生物医学和公共健康数据的统计模型；

✔️ 理解科学中严谨统计推断的必要性。

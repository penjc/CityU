# BIOS6905
## 临床试验

## **📘 简介**
本课程旨在介绍临床试验设计与分析的实际方法，包括随机化、分层、盲法设计、组序列分析及自适应设计。内容涵盖 I 期剂量探索试验、II 期初步试验和 III 期确认性试验。学生将学习使用计算工具和网络应用程序实现相关分析，掌握在医药行业和学术界设计与分析临床试验的技能。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 解释临床试验的基本概念与方法；

✔️ 理解并掌握相关技术及工具；

✔️ 阅读并理解相关文献，掌握当前最先进的工具；

✔️ 将所学技术应用于实际数据分析。

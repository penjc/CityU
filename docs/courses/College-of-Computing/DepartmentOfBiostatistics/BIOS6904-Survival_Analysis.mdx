# BIOS6904
## 生存分析

## **📘 简介**
本课程教授学生分析时间到事件数据及右删失数据的估计技术。内容包括：时间到事件数据的分布、删失和截尾类型、单变量生存曲线估计、多变量模型（如加速寿命模型和比例风险模型）的推断及其参数限制（从参数到非参数）。学生将使用 R 软件分析真实数据。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 解释生存分析的基本概念与方法；

✔️ 应用技术和方法分析真实数据；

✔️ 熟悉时间到事件数据的当前研究进展及应用工具。

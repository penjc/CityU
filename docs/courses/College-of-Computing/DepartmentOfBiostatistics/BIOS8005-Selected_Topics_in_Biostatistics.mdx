# BIOS8005
## 生物统计学专题

## **📘 简介**
本课程旨在引导研究生深入一个先进的生物统计学领域（如因果推断、基因组分析、高维数据分析、后模型选择、强化学习、生存分析或时间序列分析等）。通过全面了解基本概念、方法与理论，学生将掌握独立开展创新研究所需的技术与知识。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/BIOS8005.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/BIOS8005.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 解释高级生物统计学领域的基本概念和方法；

✔️ 掌握相关技术并深刻理解其理论基础；

✔️ 熟悉文献并了解最先进的研究工具；

✔️ 将所学技术应用于实际数据分析；

✔️ 发展独立研究技能和能力；

✔️ 有效沟通和展示研究成果。

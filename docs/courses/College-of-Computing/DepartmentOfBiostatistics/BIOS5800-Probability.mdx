# BIOS5800
## 概率论

## **📘 简介**
本课程旨在教授概率论的基本原理，为理解统计推断、多元分析、回归建模及生存分析奠定基础。课程内容包括：概率公理、随机变量、一维与多维分布函数、相关性、条件概率与密度、伪随机数生成、生存函数与风险函数、矩生成函数及特征函数、随机变量的无限序列、大数法则及多元中心极限定理。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 理解概率论的基本原理；

✔️ 构建概率模型以应用于公共健康等领域的实际问题；

✔️ 认识概率思维在数据分析中的重要性。

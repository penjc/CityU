# BIOS5802
## 生物统计高级方法

## **📘 简介**
本课程旨在为学生提供进行非连续响应（如二值或计数值）、相关观测及删失数据回归分析的工具。课程内容包括：非高斯响应的广义线性模型（GLM）、相关观测的混合效应模型与广义估计方程（GEE）、纵向数据分析以及 Cox 比例风险模型，用于生存时间数据的分析。实例均选自公共健康及生物医学领域。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 构建并拟合离散响应数据的回归模型；

✔️ 构建并拟合调整相关数据与删失观测的回归模型；

✔️ 解释回归模型的结果并将其应用于公共健康领域。

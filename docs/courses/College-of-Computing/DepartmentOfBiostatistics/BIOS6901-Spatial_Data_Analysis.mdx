# BIOS6901
## 空间数据分析

## **📘 简介**
本课程介绍处理地统计数据和点模式的科学与艺术，重点是通过设计、推断和诊断从数据中获取有意义的信息。课程内容包括地统计学（应用于流行病学）、变差函数估计、普通克里金和通用克里金、点过程理论、时空点模式等。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 选择合适的统计方法处理空间数据的相关问题；

✔️ 为空间现象构建统计模型，并使用合适的软件进行参数估计；

✔️ 评估不同方法的优缺点，并能够批判性地改进模型。

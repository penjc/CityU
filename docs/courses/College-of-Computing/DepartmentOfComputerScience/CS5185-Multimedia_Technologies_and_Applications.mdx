# CS5185
## Multimedia Technologies and Applications

## **📘 简介**
本课程提供对多媒体技术的理论和技术理解，涵盖多媒体数据表示、压缩、检索和通信等主题。学生将学习图像、视频、音频等多媒体数据的基本操作方法及其相关标准，同时探讨多媒体系统的设计和实现。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202425/course/CS5185.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/CS5185.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 解释数字多媒体数据的表示方法及其基于人类视觉和听觉感知的特性；

✔️ 分析图像、视频和音频的频域特性并识别其关键成分；

✔️ 阐释图像、视频和音频压缩标准的主要步骤，包括量化和编码技术；

✔️ 解释多媒体流式传输和实时数据传输的质量保证技术；

✔️ 应用无损和有损压缩技术处理多媒体数据。

# CS5481
## 数据工程

## **📘 简介**
本课程涵盖数据工程过程的完整生命周期，内容包括数据获取、清洗、处理、存储、管理以及应用。学生将学习高级数据工程技术，例如网络爬虫、数据库系统、数据可视化和处理算法。同时，本课程还探讨了数据管理中的关键问题，如数据质量、安全性、隐私以及联合处理，帮助学生掌握支持复杂数据工程应用的技能。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述数据工程流程的生命周期，包括数据获取、清洗、处理、存储、管理和应用；

✔️ 应用数据工程技术进行数据采集和处理；

✔️ 探讨数据管理中特定问题，例如数据质量、安全性和隐私；

✔️ 运用数据工程技术进行实际数据应用案例的分析，如推荐系统、异常检测和信息检索。

# CS6520
## Project

## **📘 简介**
本课程为学生提供探索计算机科学领域特定主题的机会。学生将应用已有知识，分析复杂问题，设计并开发具有工作原型的创新解决方案，并通过书面报告与演示展示其成果。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202425/course/CS6520.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/CS6520.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 确定并分析复杂计算机问题，提出创新的解决方案；

✔️ 通过设计与开发工作系统或应用程序实现解决方案；

✔️ 实现并评估开发的系统或应用程序以满足初始需求；

✔️ 使用不同的媒体记录和汇报系统设计、研究、实施与评估结果。

---

## **📊 评估方式**

| **评估项目**              | **权重** | **具体描述**                                                                                       |
|--------------------------|---------|---------------------------------------------------------------------------------------------------|
| **📝 Project Planning**    | 10%     | 确定研究问题并制定项目计划，包括适当的里程碑设置。                                                    |
| **📄 Project Proposal**     | 50%     | 分析问题，研究现有解决方案，并提出创新设计和解决方案。                                                |
| **🧑‍💻 Project Implementation and Evaluation** | 30% | 实现并验证设计方案，并通过评估和比较现有方案进行改进。                                              |
| **🖥️ Final Documentation and Presentation**    | 10%     | 记录项目过程并通过书面报告和口头演示展示最终成果。                                                    |

**注意**：本课程通过 100% 的课程作业进行评估。

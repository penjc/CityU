# CS5489
## 机器学习：算法与应用

## **📘 简介**
本课程旨在向学生介绍机器学习领域的算法及其应用。学生将学习如何通过机器学习算法自动从经验数据（如文本、图像、视频和传感器数据）中识别复杂模式。本课程注重从实际角度出发，教授如何实现和应用机器学习算法解决现实问题。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别并解释常见的机器学习算法；

✔️ 实现机器学习算法；

✔️ 应用机器学习算法解决实际问题；

✔️ 评估不同机器学习算法的效果，讨论其优缺点。

---

## **📊 评估方式**

| **评估项目**              | **权重** | **具体描述**                                                                                       |
|--------------------------|---------|---------------------------------------------------------------------------------------------------|
| **📝 In-class Exercises** | 10%     | 课堂练习中学习和理解机器学习算法。                                                                   |
| **📄 Assignments**         | 30%     | 实现和应用机器学习算法，分析不同算法的效果。                                                         |
| **🧑‍💻 Course Project**      | 30%     | 设计和实现一个基于机器学习算法的系统，并进行结果分析。                                               |
| **🖥️ Final Examination**    | 30%     | 测试学生对机器学习算法的理解与应用能力，考试时间为 2 小时，最低通过分数需达到考试总分的 30%。            |

**注意**：必须在期末考试和课程项目中分别取得至少 30% 的分数才能通过课程。

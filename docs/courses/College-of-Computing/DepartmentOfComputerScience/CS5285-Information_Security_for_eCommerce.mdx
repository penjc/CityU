# CS5285
## 电子商务信息安全

## **📘 简介**
本课程深入探讨信息安全在电子商务中的应用，介绍数据保护和系统保护的需求与方法。学生将学习电子商务系统设计、开发与管理中涉及的多种信息安全问题和设计策略。

- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/CS5285.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 阐释电子商务系统对数据保护的组织需求；

✔️ 分析影响电子商务系统安全性的因素；

✔️ 应用制定的策略评估电子商务系统的安全性；

✔️ 描述与电子交易、数据隐私保护和网络访问相关的法规；

✔️ 设计并分析保护电子商务系统和交易的安全机制。

| **评估项目**          | **权重** | **具体描述**                                                                                       |
|----------------------|---------|---------------------------------------------------------------------------------------------------|
| **📝 问题集 1**        | 10%     | 评估学生理解与应用课程中信息安全基本原理的能力。                                                 |

| **📄 问题集 2**        | 10%     | 学生需要分析和设计安全系统，并识别实际应用中的信息安全原理。                                      |
| **🧠 期末考试**        | 60%     | 综合评估课程所有内容，需获得至少 30% 的分数方可通过课程。

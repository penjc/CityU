# CS5182
## 计算机图形学

## **📘 简介**
本课程涵盖 2D 和 3D 计算机图形学的核心概念和算法，以及图形学技术在各个领域的实际应用。课程旨在让学生熟悉当前的计算机图形学技术，了解其在建模、渲染和动画方面的实际应用。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别并解释不同计算机图形学技术的动机与主要特性；

✔️ 设计并开发计算机图形学算法；

✔️ 评估和分析不同类型的图形系统；

✔️ 将图形处理技术应用于实际场景。

---

## **📊 评估方式**

| **评估项目**           | **权重** | **具体描述**                                                                                       |
|-----------------------|---------|---------------------------------------------------------------------------------------------------|
| **📝 期中测验**         | 15%     | 测试学生对图形学基础和算法的理解。                                                                |
| **📄 课程项目**         | 25%     | 学生需开发基于 OpenGL 或深度学习的图形学项目，并展示其设计与实现过程。                              |
| **🧑‍💻 期末考试**        | 60%     | 测试学生对计算机图形学核心内容的综合理解。                                                        |

**注意**：期末考试和课程项目均需达到总分的 30%，才能通过课程。

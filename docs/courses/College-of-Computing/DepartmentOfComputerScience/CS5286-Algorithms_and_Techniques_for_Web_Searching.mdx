# CS5286
## 网络搜索与算法技术

## **📘 简介**
本课程介绍用于网页搜索的算法与技术，包括信息检索模型、排序算法、爬虫技术和搜索引擎的实现。课程注重理论与实践相结合，帮助学生理解搜索引擎背后的核心算法。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202425/course/CS5286.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/CS5286.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 创建用于网页搜索的自动化软件代理；

✔️ 分析超链接结构并提取信息；

✔️ 使用不同的文档模型为网页创建索引；

✔️ 根据相关性对查询进行排序并评估其效果；

✔️ 创造性地应用网络搜索工具解决实际问题。

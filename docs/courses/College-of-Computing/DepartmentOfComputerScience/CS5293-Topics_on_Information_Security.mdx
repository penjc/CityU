# CS5293
## 信息安全专题

## **📘 简介**
本课程旨在为学生提供信息安全领域一系列主题的全面理解，重点关注识别实际系统中的安全威胁及其应对措施。学生将学习如何分析网络和应用的威胁，制定和评估适当的安全措施。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别并分析软件和网络应用的常见威胁和漏洞；

✔️ 分类并分析网络和系统的常见威胁和漏洞；

✔️ 提出和评估针对软件、网络应用和系统攻击的主要对策；

✔️ 识别并研究新兴计算技术和应用中的安全问题。

## **📊 评估方式**

| **评估项目**          | **权重** | **具体描述**                                                                                       |
|----------------------|---------|---------------------------------------------------------------------------------------------------|
| **📝 Assignment 1**   | 13%     | 包括对软件和网络攻击的基本理解，以及防御方法的掌握。                                                   |
| **🖥️ Assignment 2**   | 13%     | 涉及网络和系统攻击及其防御技术的理解与分析。                                                          |
| **📄 Assignment 3**   | 13%     | 针对软件与系统中的高级威胁和对策进行深入分析。                                                         |
| **🧑‍💻 Project**       | 11%     | 进行一个案例研究或详细调查，如密码学技术、安全协议的安全属性分析、命令注入攻击防御等。                  |
| **📝 Examination**    | 50%     | 总结和评估课程的整体内容。考试时间为2小时，最低通过分数需达到考试总分的30%。                                |

**注意**：必须在考试中取得至少 30% 的分数才能通过本课程。

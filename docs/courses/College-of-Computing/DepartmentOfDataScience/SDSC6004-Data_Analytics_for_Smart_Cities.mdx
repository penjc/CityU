# SDSC6004
## 智慧城市的数据分析

## **📘 简介**
现代城市依赖于将用户与基础设施连接的数据流，因此数据科学技能在智慧城市的设计和运作中尤为关键。本课程将教授适用于智慧城市数据分析的基础统计方法和机器学习算法，涵盖交通、环境、建筑能源系统及电网等领域，培养学生利用数据科学解决智慧城市问题的能力。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 阐释数据科学在智慧城市设计和运行中的原理；

✔️ 了解数据丰富化如何革新智慧城市的应用；

✔️ 针对不同智慧城市应用场景选择适合的数据科学方法；

✔️ 利用数据分析方法改进智慧城市的设计或运行；

✔️ 解释物联网在智慧城市中的作用。

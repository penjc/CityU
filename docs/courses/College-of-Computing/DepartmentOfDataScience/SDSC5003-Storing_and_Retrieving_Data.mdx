# SDSC5003
## 数据存储与检索

## **📘 简介**
本课程提供关系型数据库的知识以及 Hadoop/Spark 系统的基础介绍，包括实体关系模型、数据库开发过程原则、结构化查询语言（SQL）的使用，以及 Hadoop/Spark 的入门内容。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/SDSC5003.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/SDSC5003.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述关系型数据库设计原则和结构化查询语言语法；

✔️ 解释数据库系统中的效率问题，包括存储、索引及查询优化；

✔️ 讨论现代数据库技术，例如 JSON 和 NoSQL；

✔️ 说明 MapReduce 计算框架和 Spark 基础知识；

✔️ 运用所学知识开发数据库应用程序。

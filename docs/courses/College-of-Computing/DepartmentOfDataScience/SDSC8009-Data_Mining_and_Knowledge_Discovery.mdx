# SDSC8009
## 数据挖掘与知识发现

## **📘 简介**
本课程聚焦于数据挖掘算法和计算范式，帮助学生学习从原始数据中提取有用知识的方法。课程内容从传统数据挖掘方法开始，涵盖表格数据、图数据分析和高级算法设计，逐步扩展到基于视觉分析的高级算法及其在现实世界中的实际应用。

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述数据挖掘的基本算法；

✔️ 阐释通过数据挖掘解决问题的基础知识；

✔️ 实现课程中介绍的数据挖掘算法；

✔️ 将所学算法应用于实际问题；

✔️ 运用所学算法从实际问题中提取新颖知识。

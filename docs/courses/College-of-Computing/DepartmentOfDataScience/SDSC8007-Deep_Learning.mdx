# SDSC8007
## 深度学习

## **📘 简介**
本课程系统讲解深度学习的理论与方法，包括浅层与深层神经网络、全连接与结构化神经网络、卷积与傅里叶变换、梯度下降与随机梯度下降、反向传播与自动微分，以及深度学习算法的学习能力和网络架构设计等。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 解释深度学习的基础理论、方法和应用；

✔️ 分析和比较不同深度神经网络架构的功能与性能；

✔️ 运用标准深度学习算法分析现实世界数据集并提出解决方案；

✔️ 解决实际问题并创新设计深度学习算法。

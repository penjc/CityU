# SDSC6008
## 实验设计与回归

## **📘 简介**
本课程旨在教授学生实验设计和回归方法的基础知识，培养设计和分析物理及计算机实验的能力，强调系统化方法的价值。内容涵盖正交设计、空间填充设计、线性模型、最小二乘法、方差分析及模型选择方法，并探讨高斯过程模型在复杂非线性关系建模中的应用。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 定义实验设计和统计分析方法的类型；

✔️ 为特定物理系统设计高效的数据收集实验；

✔️ 应用模型构建与选择技术，发现物理系统中输入与输出间的关系；

✔️ 为计算系统设计高效的数据收集实验；

✔️ 应用高斯过程建模发现计算系统中高度非线性的输入输出关系。

# SDSC5001
## 统计机器学习 I

## **📘 简介**
本课程专注于统计机器学习的理论基础和基本方法，涵盖概率论和统计推断的关键概念、回归和分类的经典与前沿方法，以及无监督学习的常用方法。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202526/course/SDSC5001.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202526/course/SDSC5001.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 总结统计推断理论在机器学习中的关键概念；

✔️ 构建基本的统计回归和分类模型；

✔️ 建立核机器与正则化形式；

✔️ 描述每种学习方法中的偏差-方差权衡；

✔️ 运用所学的机器学习方法解决实际数据分析问题。

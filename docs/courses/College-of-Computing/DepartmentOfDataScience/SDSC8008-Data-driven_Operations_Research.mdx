# SDSC8008
## 数据驱动的运筹学

## **📘 简介**
本课程探讨数据驱动决策制定、风险管理和运筹学中的研究方法及其实际应用，主题包括预测与决策分析、复杂数据结构学习、不确定性下的决策优化以及机器学习结果的可解释性。课程强调方法在电子商务平台、在线市场和金融市场中的实际应用。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述数据驱动运筹学的基本原理并阐述具体方法；

✔️ 详细说明每种方法背后的关键思想及其成功的原因；

✔️ 评估现行商业和金融实践并提出改进方案；

✔️ 运用所学方法解决决策制定、风险管理和运营中的实际问题。

# SDSC6013
## 金融工程与技术专题

## **📘 简介**
本课程探讨金融服务行业中由技术创新引发的变革，包括移动支付、外汇交易、市场借贷、储蓄与投资、智能顾问和个性化保险等领域。学生将学习金融工程的基本概念及模型，通过实际案例和学术研究深入理解技术与经济驱动的变化，培养解决金融技术领域实际问题的能力。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 清晰阐述金融工程和金融技术的基础概念及其具体应用；

✔️ 详细说明每个领域背后的关键思想及其成功的原因；

✔️ 评估金融工程与金融技术的现行实践并提出改进方案；

✔️ 运用课程所学解决金融技术领域的实际问题。

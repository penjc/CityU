# SDSC8013
## 分类数据分析的统计方法

## **📘 简介**
本课程教授学生如何使用描述性统计、检验统计、统计模型和统计推断来分析分类数据，涵盖的主题包括比值比和相对风险的统计推断、多维列联表分析、广义线性模型、多类别 Logit 模型、配对数据推断以及对数线性模型。通过案例研究，学生将学习统计方法在不同领域中的应用及其结果解读。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述分类数据分析中的不同研究设计和数据类型；

✔️ 解释描述性统计和检验统计的基本原理；

✔️ 使用统计软件进行分析并拟合模型；

✔️ 正确解读模型结果并从案例研究中得出结论；

✔️ 应用所学方法分析实际分类数据集。

# SEE5211
## 环境应用中的数据分析 (Data Analysis in Environmental Applications)

## **📘 简介**
本课程教授统计方法在环境科学中的应用，涵盖概率分布、假设检验、回归分析和方差分析等。学生将学习如何分析环境数据并探索其与环境问题之间的关系，包括使用 Python 进行编程和报告撰写。

🔗 **相关链接**
- [课程页面](https://www.cityu.edu.hk/catalogue/pg/202425/course/SEE5211.htm)
- [课程大纲 PDF](https://www.cityu.edu.hk/catalogue/pg/202425/course/SEE5211.pdf)

---

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 掌握基本统计方法概念；

✔️ 运用统计方法描述环境数据；

✔️ 分析数据并发现其与环境问题的联系；

✔️ 创造性地解释环境科学中的基本物理过程。

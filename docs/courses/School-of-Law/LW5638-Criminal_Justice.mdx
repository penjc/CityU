# LW5638
## 刑事司法 (Criminal Justice)

## **📘 简介**
本课程提供对香港刑事司法系统的深入分析，内容包括法律执行、审判和惩戒制度，重点探讨刑事司法的主要参与者、机构和程序。本课程不仅关注刑事司法的具体操作，还将探讨其背后的哲学和道德问题，反思其在香港及其他司法管辖区的应用。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述并分析香港刑事司法系统的组成、操作及其主要机构；

✔️ 反思并分析刑事司法系统中的内在张力和挑战；

✔️ 批判性评价香港及其他司法管辖区刑事司法系统的优势与不足。

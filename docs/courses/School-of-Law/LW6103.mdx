# LW6103E
## 公司法 (Company Law)

## **📘 简介**
本课程的主要目的是使学生了解香港公司法的法律原理和规则，涵盖公司法的基本原则、公司治理、公司管理结构、股东与董事之间的关系等内容。课程分析公司法的理论基础及其政策驱动因素，帮助学生理解如何在公司运营中解决相关法律问题。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 描述并解释公司法的基本原则；

✔️ 通过案例分析、法规解释，清晰、逻辑、连贯地解决公司法相关法律问题；

✔️ 研究、分析并批判性评估公司法中的法律原则，并理解其如何在平衡商业目标和社会目标中发挥作用。

|---------------------|---------|---------------------------------------------------------------------------------------------------|

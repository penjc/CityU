# LW6119E
## 宪法法 (Constitutional Law)

## **📘 简介**
本课程旨在帮助学生深入了解香港特别行政区基本法及其与中华人民共和国宪法的关系。课程将探索“一国两制”原则，分析香港特区与中央政府之间的法律关系，讨论香港的行政权、立法权和司法独立。通过案例分析，学生将学习如何应用宪法和基本法的原则解决实际法律问题。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 解释宪法法的基本概念和“一国两制”原则；

✔️ 了解香港特别行政区与中央政府的关系及香港政府的运作；

✔️ 分析和理解宪法案例，应用宪法原则解决实际问题；

✔️ 批判性地分析和解决基本法实施过程中可能出现的重大问题和冲突。

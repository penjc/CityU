# LW6115E
## 中国知识产权法选论 (Selected Issues on Chinese IP Law)

## **📘 简介**
中国的知识产权法是一个快速发展的领域。本课程将探讨中国知识产权法的独特性，尤其是在全球化和本土化背景下的发展与挑战。学生将了解版权法、专利法、商标法等领域在中国的法律框架，尤其关注跨境知识产权案件的处理。通过研究中国知识产权法的最新发展，学生将具备分析和解决跨境知识产权问题的能力。

## **🎯 学习目标**

完成课程后，学生将能够：

✔️ 识别并解释中国知识产权法中的关键法律原则；

✔️ 分析中国知识产权法的结构和理论；

✔️ 将中国知识产权法应用于跨境案例，为香港与内地之间的知识产权问题提供法律建议。
